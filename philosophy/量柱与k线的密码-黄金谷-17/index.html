在下跌的过程中，可以连续出现下跌缩量的情况，经常会缩量到1/3或1/2，当这个现象出现的时候，也不需要急于进场。因为在这个过程中，主力资金在不断的试探市场的底部，这个过程或许会很漫长，或许跌幅会比较大。如果贸然冲进去那可能会比较难受，因为下跌可能还才刚刚开始，下跌的时间和幅度并不能准确的预测出来。这个时候选择等待是最好的方式，当股价再也放不出量了，也到不了新低的时候，或许就是股价到达真正市场底部的时候。当市场连续出现百日低量的时候，股价大概率是到了真正的底部，这个时候如果有耐心的资金可以尝试介入。因为我们所说的黄金梯黄金踏步就要随之而来了，新的一轮上涨就会开始孕育，重新上涨的故事又重新开始了。

📊 形态定义：黄金谷底部信号——连续百日低量，股价不再创新低。

【黄金谷——底部形成过程】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

量柱层（对应过程）：
   ┃         ┃
   ┃    ┃    ┃     ┃
   ┃    ┃    ┃     ┃     ┃
   ─┴────┴────┴─────┴─────┴─────────────
   下跌期   缩量期   百日低量
   （放量） （逐步缩） （缩到极致）
   └──────────┬──────────┘
   黄金谷的底部信号：
   ① 连续出现百日低量
   ② 股价不再创新低
   ③ 下跌动能耗尽
   → 黄金谷形成！此时可尝试介入 💰

所以这个谷底就可以称之为黄金谷，因为在这个位置介入的资金大概率能赚钱，但是通过上一波的下跌，很多资金都已经不敢介入了，也有不少在高位进场的资金在这个位置纷纷割肉出逃，这进入了主力资金的陷阱。

随着股价逐渐通过黄金踏步到达左侧下跌开始的位置后，因为左侧有大量的套牢盘，而且随着股价的上涨，成交量也会不断放大，抛压也会逐渐放大。在这个位置，如果想继续拉升，显然是不太现实的，必须要把左侧的套牢盘震荡出去，而且此时的跟风资金也有很多，同时也需要甩出去。所以在这个位置会形成一个下跌凹口，下跌的幅度不会太大，它的目的是需要把跟风资金和左侧的套牢盘甩出去。

所以在这个凹口位置，主力资金会让突然大幅下跌，跌破前一日开盘价，而且量能也会放出来，但是下跌的幅度就只到左侧的压力位。下跌的幅度不会太大，不过这只是相对于上涨的幅度来说的，如果对于短利来说下跌的幅度也会比较大，会10%将近20%，甚至更多。但它到达一定的位置之后，就会缩量1/2或1/3，甚至更低，在这个时候想抛的资金已经全部抛出去了，那谁在接盘呢？就是主力资金。在这个位置抛压已经很小了，所以下跌的动能已经不大了，主力资金用少量的资金就可以维护住股价，同时也可以吸收到一定的筹码。在这个过程中，也可能会有一些小小的上涨或下跌，只不过主力资金用来拉升或打压股价来吸收筹码而已，通过这种方式，让跟风资金死去活来。这样震荡的时间也不会太长，可能成交量就会达到一个新的地量，当出现这样的情况时，我们必须要睁大眼睛果敢介入，因为上涨就在眼前。

📊 形态定义：左侧大凹口（吸筹）vs 右侧小凹口（洗盘）。

【左侧凹口 vs 右侧凹口对比】

左侧大凹口（吸筹）：
   ┌─────────────────────────────┐
   │ ① 主要任务：吸筹              │
   │ ② 时间：长                   │
   │ ③ 成交量：小（悄悄吸筹）        │
   │ ④ 凹口大小：大                │
   │ ⑤ 底部成交量：地量            │
   └─────────────────────────────┘

右侧小凹口（洗盘）：
   ┌─────────────────────────────┐
   │ ① 主要任务：洗掉左侧套牢盘     │
   │     + 洗掉跟风资金            │
   │ ② 时间：短                   │
   │ ③ 成交量：大（快速洗盘）       │
   │ ④ 凹口大小：小               │
   │ ⑤ 底部：不能低于左侧最低点     │
   └─────────────────────────────┘

在上涨凹口位置的左侧，有中到大阳线突然拉升，但也有中到大阴线突然跌下来，之所以会有这样大的动静，就是主力资金想上涨了，可是手中的筹码却还没有达到一定的程度。通过暴力拉升的方式快速吸收到大量筹码，伴随着股价上涨，成交量也会放大，会形成高量柱或者倍量柱。下跌的初始大阴线和量柱也会比较大，因为下跌会比较突然，所以跟风资金纷纷选择割肉，主力资金在这个位置也吸收到了大量筹码。上涨凹口还有一个特点就是在凹口的底部成交量十分小，而且在右侧位置马上有一根大阳线突然拉升，他的目的就是让那些跟风资金跟不上来。因为这样的大阳线拉升十分突然，跟风资金都还没有反应过来，股价就上去了，这就是主力资金故意的，避免让跟风的资金跟上来。

其实这样的上涨凹口还要比左侧的上涨凹口要小很多，但是因为左侧的上涨凹口已经沉淀了大量的筹码。左侧这个凹口是收集筹码的，当然要大，收集时间也更长，因为这个谷底它的成交量不大，也就是主力资金在悄悄的吸筹码，这样的时间当然需要比较长。所以造成了凹口当然也比较大，右侧那个凹口最主要的任务是洗掉左侧密集筹码的压力，同时也洗掉上涨这几天所带来的跟风资金。所以这个凹口的主要任务是洗而不是吸，所以当然幅度不会太大，时间也不会太长，要完成这样的任务，成交量相对左侧的凹口底部来说会更大一些。所以右侧凹口的时间会更短，成交量会更大，跌的速度也会比较快，但上涨的速度同样很快。

右侧凹口的最低价格不能低于左侧凹口的最低价格，如果低于左侧的最低价格，那左侧就不是吸筹了，那上涨的趋势就没法形成，那就不是金坑了，而是粪坑了，因为是一浪比一浪更低。

📊 形态定义：金坑 vs 粪坑——右侧底部高于左侧=金坑，低于左侧=粪坑。

【金坑 vs 粪坑】

✅ 金坑（右侧底部高于左侧）：
        ┃
        ┃              ★ 右侧底部
        ┃              ────  ┃
        ┃  ★ 左侧底部   ┃  ┃  ┃
        ┃  ────  ┃    ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃ ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴┴──┴──┴──┴─────────────
   左侧底部（低）    右侧底部（更高）
   └──────────┬──────────┘
   右侧底部 > 左侧底部
   → 一浪比一浪高
   → 上涨趋势形成 ✅
   → 这是金坑！💰


❌ 粪坑（右侧底部低于左侧）：
        ┃
        ┃  ★ 左侧底部
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃         ★ 右侧底部
        ┃  ┃  ┃  ┃  ┃         ────  ┃
        ┃  ┃  ┃  ┃  ┃         ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────┴──┴──┴──┴─────────────
   左侧底部（高）             右侧底部（更低）
   └──────────┬──────────┘
   右侧底部 < 左侧底部
   → 一浪比一浪低
   → 下跌趋势延续 ❌
   → 这是粪坑！💩

其实通过黄金梯的上涨，咱们可以画出上涨的趋势线，按照传统的上涨趋势线的画法，就是分别按照最低价格和最高价格连成一条线。这样就会形成两条线，一个是最低价格连成的趋势线，一个是最高价格连成的趋势线，如果跌破最低价格连成的趋势线，那么趋势就会反转，下跌即将到来。如果突破了上涨趋势线，那么新的上涨趋势又会形成。这个是按照传统的k线价格来形成的两条趋势线，但是我们既然研究量柱，我们就按照量柱的高度来画两条不同的趋势线。可想而知，按照最低量柱不同点连线形成一条线，同时也按照最高量柱的不同点形成另一条趋势线，这样也构成跟k线价格最低点和最高点连线有点类似的趋势线。甚至两种方法所形成的趋势线，有的时候竟然能重合，说明不同的方法，有些本质的内容还是一样的，只不过表现形式不一样而已。

📊 形态定义：三条趋势线——最低价线、最高价线、量柱相对低点线。

【三条趋势线——价格两条 + 量柱一条】

价格层趋势线：
                    ▲ 上涨趋势
        ┃           │
        ┃     ┃     │  ← 最高价连线（趋势线②）
        ┃     ┃  ┃  │
        ┃  ┃  ┃  ┃  ┃  ← 最低价连线（趋势线①）
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────────
   ① 最低价连成的趋势线
   ② 最高价连成的趋势线
   ┌─────────────────────────────┐
   │ 跌破① → 趋势反转，下跌将至 │
   │ 突破② → 新上涨趋势形成     │
   └─────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

量柱层趋势线（创新）：
   ┃           ┃
   ┃     ┃     ┃  ┃
   ┃     ┃     ┃  ┃  ┃
   ─┴─────┴─────┴──┴──┴─────────────
   ③ 量柱相对低点连线（第三条趋势线）
   └──────────┬──────────┘
   第三条线的独特价值：
   ① 从价格角度无法连出这条线
      （它在最高价和最低价之间）
   ② 从量柱角度可以连出来
      （量柱相对低点对应K线最低价）
   ③ 走向与①②大致保持一致
   ④ 可实时观测量柱长短变化
      → 观测人气变化 👀

我们现在所需要研究的就是，在量能最大的位置画一根趋势线，在量柱最小的位置也会画一根趋势线，这样形成一个量柱的范围区间，在这种情况下，绝大部分情况下价格也会在这个区间运行。如果仅仅是研究这两条趋势线，那我们就和价格的最低点连线和价格的最高点连线形成的趋势线没有太大的区别了。现在我们不满足于此两条趋势线，如果仅仅还是这两条趋势线，那我们就没有独特性了。我们需要创新性的再拉出一条线，这条线如果仅仅以k线的价格是很难连出来的，因为这条线在最低价格和最高价格两条趋势线之间，从价格的角度当然不能连出来。可是以量柱的方式还是可以连出来的，在最高趋势线和最低趋势线之间，量柱还有一个相对的低点。我们可以把这个相对低点的量柱锁定的k线价格最低点把它连起来，这样也会形成一个趋势线，这根趋势线的走向，能够和最高点和最低点所连成的趋势线大致保持一致。这就是我们从量柱的角度跟从价格的角度所不同的地方，也是量柱研究的独特性。那么中间这个线究竟能起到什么作用呢？咱们可以实时观测在这个区间量柱的变化情况，通过量柱的长短变化，可以观测出人气的变化。
