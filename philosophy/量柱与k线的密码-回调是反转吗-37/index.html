而且如果一旦确认是支撑的时候，主力资金会快速拉升股价，让跟风资金跟不上，迅速摆脱主力资金的成本区域。其实这就像一浪一浪的波浪一样，它的第一浪或许会比较高，第二浪会比第一浪高度降低，直至它的浪高越来越低。这样其实也是一个上涨动能衰竭的过程，它上涨的过程中，一开始因为没有什么抛压，所以上涨的高度比较高，越往上走的抛压越来越大，理所当然的幅度也越来越低。不过这也可能是表面现象，主力在这个过程中可能在积蓄新的上涨动能，它也需要让这些跟随资金甩下车，所以随之而来的会有一波回调，但这个回调也要看它的抛压大小，有可能主力资金在推动新的上涨波浪。从小的周期来看，或许这一波上涨已经结束了，从更大的级别来看，它正在积蓄更多的上涨动能，主力资金在上涨的过程中也同时也在收集筹码。同时在回调的过程中，也在收集更多的筹码，有一点点筹码就让那些跟风的资金出逃，这样可以一举两得。

📊 形态定义：波浪上涨——第一浪高，后续抛压增大涨幅收窄，回调蓄势。

【波浪上涨——第一浪高→抛压增大→涨幅收窄】

        ┃         ┃         ┃
        ┃     ★第一浪  ┃  ★第二浪 ┃  ★第三浪
        ┃  ────  ┃     ────  ┃     ────  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴──┴──┴──┴──┴─────────────
   第一浪  回调  第二浪  回调  第三浪
   （高度高）     （降低）       （更低）
   └──────────┬──────────┘
   第一浪：没有什么抛压 → 涨幅高
   后续：抛压越来越大 → 涨幅收窄
   但回调可能是表面现象 → 主力积蓄新动能
   小周期看上涨结束 → 大周期看正在蓄势 🔄

在回调的过程中可能会出现一浪低过一浪的现象，但是如果出现了抛压逐渐降低，但是没有跌破起涨点的时候，我们就需要高看一眼了。因为在这个过程中，主力资金就是在蓄势，现在只是刚刚完成了第一波浪的上涨和回调，紧接着的第二波上涨即将开始。那又是重复昨天的故事了，重新突破左侧的高点，形成第二波的上涨浪，因为左侧的位置积蓄了大量的筹码，跟风资金也很少了，所以上涨的时候抛压也会偏少，所以上涨可能比第一波会更猛。

📊 形态定义：回调判断——不跌破起涨点则蓄势，第二波更猛。

【回调判断——不跌破起涨点=蓄势】

价格层：
        ┃
        ┃     ★第一波上涨
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ★回调
        ┃  ┃  ┃  ┃  ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ★第二波上涨更猛
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ────  ┃

   └──────────┬──────────┘
   回调中抛压逐渐降低
   但没有跌破起涨点
   → 主力在蓄势！
   → 第二波上涨即将开始
   → 左侧筹码已消化，跟风资金少
   → 上涨可能比第一波更猛 🚀

通过咱们以上的讨论，我们可以来理一下思路，什么样的量柱形式，什么样的k线组合，会形成比较好的上涨几率。其实我们之前也说过上涨的时候放量，那说明主力资金可以在积极吸货，不管他是主动还是被动的，但总是在吸货。最重要的一个观察点就是，在看他回调的时候有没有放量，如果回调的时候也放量，说明主力自己他也不太看好了。这个时候又要探测新的底部，重新积蓄上涨的动能了。还有一种可能性，就是在下调的过程中缩量，最简单而言就是缩了1/3或1/2。这种情况下，就是主力资金在收集筹码，用一点点筹码就打出了阴线的恐慌形态，它最主要的目的是甩掉跟风而不是真的下跌。所以出现价格往上涨，阳线量柱逐步升高，但是下跌的时候缩量，那么这种量柱与k线的形态组合就是好的上涨形态组合，这种组合上涨的概率极大，如果反复出现这样的形态组合，那么上涨会更加猛烈。

📊 形态定义：最好的上涨形态组合——上涨放量+回调缩量。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   ┃         ┃                    ┃
   ┃    ┃    ┃     ┃              ┃    ┃

   （吸货）    （主力收集筹码）      （强势）
   └──────────┬──────────┘
   上涨放量 + 下跌缩量 = 好的上涨形态组合 ✅
   回调缩量 → 主力用少量筹码制造恐慌
   → 目的是甩掉跟风，不是真的下跌
   → 反复出现 → 上涨更猛烈 🚀

其实在上涨的过程中会形成踏步，这个踏步有的是向下回调的踏步，有的是水平踏步，那这个踏步在某个价格区间，肯定形成了筹码密集峰。通过这种方式，不断打牢向上攻击的基础，其实这个踏步也就是主力资金在某个阶段的成本，在这个踏步位置不断吸筹洗筹，通过这种方式吸收了更多的筹码，同时也洗掉了不稳定的筹码。在这样踏步位置，不断的吸筹洗筹，主力资金便会掌握大量的筹码，到一定的程度时市场上几乎没有什么抛压了，这时候拉升更加容易了，或许主升浪出来了。

📊 形态定义：三种踏步形态——向下回调/水平整理。

        ┃
        ┃     ┃
        ┃     ┃     ┃
        ┃     ┃     ┃     ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ← 水平震荡

踏步的意义：
   ┌─────────────────────────────┐
   │ ① 形成筹码密集峰             │
   │ ② 不断打牢向上攻击的基础     │
   │ ③ 主力阶段的成本区域         │
   │ ④ 吸筹+洗筹（吸收筹码/洗掉不稳定筹码）│
   │ ⑤ 掌握大量筹码→抛压消失      │
   │ ⑥ 拉升更容易→主升浪 🚀      │
   └─────────────────────────────┘

其实我们在这个市场上就是要跟踪这样有计划有策略的主力资金，它是稳扎稳打，不过他们在一些关键位置上也是很凶狠的，要么是拉的很厉害，要么是跌的很厉害。不管是场内场外的资金看的都会目瞪口呆，因为这完全出乎绝大多数人的预料，其实主力资金在操盘的过程中也是吸引散户资金入局出局的过程。就像写一部小说的时候，先让读者感觉有趣，然后又如让读者动情动心，最后不能自己，深陷其中。

其实股票在主力资金吸筹的时候，股票并没有很大的涨幅，并不能吸引散户资金入场，只有极少数资金在这个位置可能会入场。所以这个时候对于资金来说是平淡无奇，可进可出的，并不能吸引大量的资金进入，可是主力资金突然来一个拉升，就会吸引大量的散户资金跟风入场。资金入场的依据是什么？是看到了赚钱效应，这样一拉升起来，就已经把人的贪欲勾引出来了，当然能吸引大量资金入场。就像写小说的时候，作者写了一个有趣的故事情节，吸引了读者，读者便把头埋进书里面了。但是如果按照之前的情节发展，读者并不会感觉很有趣，毕竟要出乎他的预料之外，才会让他欲罢不能。同样的道理，在股市中也会出现，所以在股市中上涨并不是唯一的主旋律，还会有下跌上涨之后的回调，同样会让很多人刻骨铭心，痛不欲生。不过股市和看书是两回事，看书并不能引起经济上的任何损失，只不过是会引起情感的共鸣，会有一种情绪价值，而在股市中会造成经济上的亏损，这种痛苦可不是一般的，它是伴随着经济损失和精神刺激。所以在股市中的痛苦远大于在其他领域的精神损失，因为在股市中同时伴随着经济损失，它可能会让操作者痛不欲生，甚至扇自己几个耳光，骂自己笨，骂自己蠢，可是这也不能解决任何问题。

只能做的就是从失败的操作中总结规律，在操作任何一只股票之前，首先要分析这支股票的动向，只有抓住了这只股票的规律之后，才有可能会赚钱，如果仅仅凭借拉升，就进入一只股票，那大概率是进去做接盘侠。

在之前我们曾经讨论过，如果要上涨就要突破左侧的阳线开盘价，因为在下跌过程中，主力资金或者其他资金都曾经做出过反抗，只不过绝大部分反抗都失败了。那么在这样的位置肯定是主力资金刻意为之的，这个位置也包含着主力资金的成本，所以要改变下跌的趋势，主力资金必须达到这样的k线位置，才有可能给别人信心。

📊 形态定义：突破左侧阳线开盘价——改变趋势的关键信号。
