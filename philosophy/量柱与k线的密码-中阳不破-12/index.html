第12章 中阳不破

结合之前我们所了解到的，出现一个大阳线，但是量柱或者是平量，或者是缩量，也可能是高量，这些都不是最重要的。最重要的是后续三天的价格，不要跌破这个阳线的收盘价，如果实现了这个目标，再加上后续呈现缩量状态，可以很明显的预测到，这个股价将会出现上涨。其实原因咱们之前也分析过很多次了，为什么会缩量，代表筹码的持有者不认可这个价格，不愿意卖出。也或许是大部分筹码都在主力自己手上，他们也不会去卖出，因为价格还没达到他们的心理价格。股价为什么会上涨？它是什么原因呢？代表有人想买，可是股价上涨，量能却放不出来，那后面会出现什么结果呢？其实就是价升量缩，价格往上涨就是代表有人买进才会往上涨，有不少人想买进股票，但是却拿不到货，必然就是股价会不断上涨。那涨到什么时候为止呢？涨到很多人觉得价格已经达到了心理价格为止，所以这样的量柱出现就是一个重要的量柱，可能是黄金柱，有可能是元帅柱，也可能是将军柱。

📊 形态定义：中阳不破——阳线后三天不破收盘价+缩量=看涨。

【中阳不破——三天不破收盘价】

        ┃
        ┃  ★ 大阳线
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 后续三天价格
        ┃  ┃  ┃  ┃  ┃    不跌破阳线收盘价
        ┃  ┃  ┃  ┃  ┃    量能不断缩小
   ─────┴──┴──┴──┴──┴───
   阳线  T+1 T+2 T+3
   （收盘价）  （缩量）
   ┌─────────────────────────────┐
   │ 条件：① 后续三天不跌破收盘价 │
   │       ② 呈缩量状态          │
   └─────────────────────────────┘
   → 筹码惜售/主力控盘
   → 价升量缩 → 必然上涨 📈

不管是哪种量柱，都是主力资金极力打造的一个重要量柱，他需要稳住自己的筹码来测试市场的抛压。所以就会拉一点看市场的抛压怎么样，如果第三天之后市场还没有什么抛压，主力资金就会认为之前的打压吸筹已经有效果。所以他后续可能就会来了一波拉升，一个阳线之后续观察三天量能情况，可以看出主力资金的运作逻辑，也可以看出主力资金的意图和实力。在这三天期间，主力资金也可能在悄悄吸筹，并且价格一般不会低于阳线的收盘价，如果普通的投资者在此时介入，可能将会迎来一波坐轿子的乐趣。

但不是所有的将军柱或元帅柱都能有很好的涨幅，它可以根据量能分成不同级别的，有的级别过低，它往往支撑不起一波涨幅。有的人认为会高量柱或倍量柱出现之后才会有大行情，但经常就不是这种情况，有的量柱高度并不高，但是它后续也有大行情。所以量柱的高度并不是判断后续行情走势的唯一标准，它需要有天时地利人和的要求。

📊 形态定义：将军柱→黄金柱→元帅柱，三级台阶，逐级增强。

【将军柱 → 黄金柱 → 元帅柱】

将军柱（基础级别）：
        ┃
        ┃  ★ 将军柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 后续三天价格
        ┃  ┃  ┃  ┃  ┃    未跌破阳线收盘价
   ─────┴──┴──┴──┴──┴───
   → 基础级别的支撑柱


黄金柱（中等级别）：
        ┃
        ┃  ★ 黄金柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 后续三天不破收盘价
        ┃  ┃  ┃  ┃  ┃    量柱不断缩小
   ─────┴──┴──┴──┴──┴───
   → 三天不破 + 缩量 = 黄金柱 ⭐


元帅柱（高级别）：
        ┃
        ┃  ★ 元帅柱
        ┃  ────  ┃  ← 跳空高开
        ┃  ┃  ┃  ┃  ┃  ← 后续三天不破
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴───
   → 跳空高开 + 三天不破 + 缩量
   → 最强级别 👑

有可能出现一个阳线之后价格并不会收于收盘价上方，在这个时候，它可能仅仅是一个将军柱，但它后续可能又会出现一个阳线，之后的价格高于收盘价，就可能会升级为黄金柱。像这种逐步合力起来的量柱才具有参考价值，因为它的上升不是单个量柱造成的，而是由多个这样的将军柱或黄金柱共同打造的。这样多个重要级别的量柱合力打造出一个趋势，在这种情况下，往往上涨的幅度会比较大，毕竟打造出这样的趋势，这些资金也需要时间。如果仅仅是一点点涨幅，并不能覆盖它的成本，所以后续的涨幅或许会很可观。

所以要有多个黄金柱连续形成趋势之后，上涨的概率高很多，这几个黄金柱距离不要太远，这样才会有连续性。如果单个的黄金柱出现，或许虽然也有上涨，但毕竟独木难支，这样或许是短期资金所为，来不及建仓，就拉一把就走人。所以出现了黄金柱并不值得惊讶，如果连续出现几个黄金柱，这才需要重视，因为几个将形成向上走的趋势。

逻辑推理：单个黄金柱 vs 多个黄金柱连续——独木难支 vs 趋势形成。

所以我们在观察股票的时候，其实就是要抓住关键的量柱，通过分析这些至关重要的部分，就可以分析出庄家的运行逻辑。在不同位置的关键环节中，含义也不一样，如果主力资金在错误的位置，去发动行情，失败也是显而易见的。这个市场上不会怕钱多，如果不掌握股市运行的规律，再多钱都也是会打水漂的。所以在不同位置的黄金柱含义也不一样，有的是在止跌位置出现，在这样的位置出现，就是股价不能跌破这个黄金柱的开盘价。这个位置的柱子的含义一个是止跌，第二个也是有主力护盘的意味，主力资金在这个位置不想再往下跌了。当震荡探底之后，又会出现一根黄金柱，但后续三天的价格不会跌破这个柱子的收盘价时，那就是更为珍贵的黄金柱了。其实这个黄金柱的出现，也就是确认前一个黄金柱止跌成功。如果说前一个位置可能还会止跌失败的情况发生，那这个位置出现在这个环境中，成功的概率就会高很多。后面或许还会出现黄金柱，那真的是往上涨的黄金柱了，所以说第一个出现黄金柱并不是最佳的介入时机，往往第二个和第三个黄金柱出现的时候，介入成功的概率会高很多。

📊 形态定义：黄金柱的三个位置——止跌→确认→上涨，介入时机依次递进。

【黄金柱的三个位置——介入时机】

位置一：止跌黄金柱（探底）
        ┃
        ┃  ★ 止跌黄金柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 不能跌破开盘价
        ┃  ┃  ┃  ┃  ┃    主力护盘，不想再跌
   ─────┴──┴──┴──┴──┴───
   ⚠️ 可能止跌失败 → 不是最佳介入时机


位置二：确认黄金柱（成功率提高）
        ┃
        ┃  ★ 确认黄金柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 三天不破收盘价
        ┃  ┃  ┃  ┃  ┃    确认前一个止跌成功
   ─────┴──┴──┴──┴──┴───
   ✅ 介入成功率高很多


位置三：上涨黄金柱（主升浪）
        ┃
        ┃  ★ 上涨黄金柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ← 真正的上涨黄金柱
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴───
   🚀 最佳介入时机
   └──────────┬──────────┘
   第一个黄金柱 → 观望
   第二个黄金柱 → 可介入 ✅
   第三个黄金柱 → 最佳介入 🚀

其实股价往上走的时候，还有一个规律的核心，就是会出现高量柱或者是倍量柱，之后一天会出现缩量柱，缩量至1/3或者1/2。并且这个位置距离前一个高点的位置很近，主力资金在这个位置出现的时候，那就是会出现到大阳的预兆。还有在股票拉升的时候，有时会出现假阴真阳线，如果第二日股价突破了阴线实体的一半，或许还是继续看涨。当有个k线过了左侧高量柱的对应股价高点时，观察它的量柱大小，如果是平量或者缩量，没有放出量来，那后续还是看涨。说明主力资金的筹码锁定的很好，大量的筹码在主力自己的手上，所以才没有什么抛压。

📊 形态定义：假阴真阳突破一半 + 轻松跨前高——两个实战判断信号。

【假阴真阳 + 轻松跨前高】

信号一：假阴真阳突破一半
        ┃
        ┃  ★ 假阴真阳（高开低走）
        ┃  ────  ┃
        ┃  ┃  ┃  ┃     ← 第二日股价
        ┃  ┃  ┃  ┃        突破阴线实体的一半
   ─────┴──┴──┴──┴───
   假阴线  突破一半
   → 继续看涨 📈


信号二：轻松跨过前高（缩量/平量）
            ┃
            ┃  ★ 跨过前高
   前高 ────   ┃
   位置  ┃  ┃  ┃  ┃  ← 平量或缩量
        ┃  ┃  ┃  ┃    没有放出量来
   ─────┴──┴──┴──┴───
   前高  跨过前高
   → 筹码锁定很好
   → 主力手上大量筹码
   → 后续继续看涨 📈

其实可以观察黄金柱的位置和价格，可以判断出主力资金想拉的幅度，当出现股价轻松跨过前高的位置的时候，如果量能没有放出来，那么可以预测它的涨幅将会翻番。不过这仅仅是预测而已，实际上会不会发生那还不一定呢，如果后续还是没有出现黄金柱，那需要在特定的位置离场。在股价运行的途中，需要特别观察假阴真阳线，当出现这样的现象时，往往会放出巨量，呈现为高量柱或者是倍量柱。
