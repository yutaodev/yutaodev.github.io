很多人操盘喜欢用5日均线来操盘，在均线上方一直拿着，一旦跌破均线就离场，这种方式有的时候还是挺好用的，特别是在强势的市场中，这种方法还是比较方便的。如果是强势的股票，这样操作是没有任何问题的，但是如果对于不选择强势的，股价很容易跌破5日线，但并没有改变原有的趋势。如果依据5日线来操作，可能会失去很多机会，所以这就看个人的选择，如果需要做短线的，需要做强势股的，那当然要用5日线，如果选择比较温和的，那这种方式可能就不大合适。

【5日线操盘——强势股 vs 温和股】

✅ 强势股（5日线操盘有效）：
       价格
        ┃
        ┃     ★股价
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃    ← 沿着5日线上涨
        ┃  ┃  ┃  ┃  ┃      不轻易破线
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──  ← 5日线
   → 强势股：5日线操盘好使 ✅


⚠️ 温和股（5日线操盘易失误）：
       价格
        ┃
        ┃     ┃     ┃
        ┃  ┃  ┃  ┃  ┃  ┃    ← 股价反复破5日线
        ┃  ┃  ┃  ┃  ┃  ┃      但趋势未变
        ┃  ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──·──  ← 5日线
   → 温和股：容易错失机会 ❌

很多人说，当沿着5日线长期之后，如果离5日线距离大约10%以上，那说明现在有很多问题了。他们的意思很明显，就说明这个时候短线的获利盘很重了，就需要离场。其实这种想法也是比较狭隘的，不一定到了10%就需要离场，最主要是看有没有资金做承接。一些强势的股票或者是热点的龙头，这个时候反而是上车的机会，所以说涨幅距离5日线，不能当做判别是否离场的唯一依据。是需要结合其他因素一起考虑的，这个股是否有热点概念，是否有资金承接，大盘环境怎么样，我们正好一起来观察。不过股价距离5日线的距离过远确实是不好的事情，因为短线的获利盘太多，如果一旦资金承接不够，回调将随时到来。如果从概率上来说这种说法也是对的，因为大盘有这么多股票，被资金青睐的个股也只是极少数，在大多数的股票来说，距离均线距离过长，回调是它们唯一的选择。这个在分时图上的表现也很明显，如果距离均线的位置过远，这个时候离场往往是一个不错的选择，只有极少数股票能走出来，不能凭借极少数股票走势来判定绝大多数。

不过这个时候不光股价距离5日线大于10%以上，同时还有一个很显著的特征就是它的量能放的很大，往往比前一日高出了三四倍，这个时候往往具有出货的嫌疑了。我在一些其他的资料上看到过也有过类似的说法，有的时候单倍量柱那倒是一件好事，如果是前一天量柱的好几倍，那就需要观察了。这个时候如果是底部第一根量柱是前一天的好几倍，可能还会比较健康。如果上涨了一段时间，突然出现了量柱是前一日的三倍以上，这个时候很大概率就是出货了。

【股价远离5日线+量能判断】

条件：股价距离5日线>10%

       价格
        ┃          ★股价离5日线>10%
        ┃          ┃
        ┃          ┃
        ┃       ┃  ┃
        ┃    ┃  ┃  ┃
        ┃ ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──  ← 5日线

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

量能判断：
   ┌─────────────────────────────┐
   │ ① 量能比前一日高出三四倍     │
   │    → 出货嫌疑 ⚠️            │
   │ ② 底部第一根高量柱（几倍）   │
   │    → 可能健康 ✅             │
   │ ③ 上涨途中突然巨量（3倍以上）│
   │    → 大概率出货 ❌           │
   └─────────────────────────────┘

   离场判断标准：
   ① 距离5日线>10%
   ② 量能放大三四倍
   ③ 两者同时满足 → 大概率出货了 💸

从上面的分析，我们可以得出这样一个结论，从5日线判断是否离场，可以从两方面来观察，一方面是与5日线的距离，第二个是量能的变化情况，如果两者都达标，那么大概率都是出货了。

中国里面有一句古话就是3岁看大，7岁看老，这句古话放在股市中也是一样。当一只股票它在吸筹的阶段不破5日均线，那么大概率在后期的走势也不破10日均线。也就是这样一个意思，如果前期的基础打的很牢固，那么后期的涨幅也很可观的。就像一个孩子一样，如果在小时候就表现出了惊人的天赋，再加上努力，后期他能取得的成就，那也远超一般的人。

【3岁看大7岁看老——前期预示后期】

吸筹阶段（前期）：
       价格
        ┃
        ┃     ★吸筹阶段
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃    ← 不破5日均线
        ┃  ┃  ┃  ┃  ┃      基础牢固
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──  ← 5日线
   └──────────┬──────────┘
   吸筹阶段不破5日线 → 基础牢固

后期走势：
       价格
        ┃
        ┃     ★主升浪
        ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃    ← 大概率不破10日线
        ┃  ┃  ┃  ┃  ┃      涨幅可观
        ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──  ← 10日线
   └──────────┬──────────┘
   前期基础打牢 → 后期涨幅可观 🚀

大家有没有发现这样一个规律，就是在上涨的时候，往往长期均线的角度小和短期均线的角度大。这样也是符合自然规律的，因为短线均线角度大，是因为短线现在涨幅很大，长期均线往往是通过短期的股价拉动的，它转变的时间比较长，所以当然角度也比较小。所以在上涨的时候就往往会看出这样一个特点，各条均线好像形成一个喇叭口一样，越往上走，喇叭口张得越大，长期资金获利也很多。

【喇叭口形态——短期角度大，长期角度小】

       价格
        ┃
        ┃    ★喇叭口张得越大
        ┃    ┃
        ┃    ┃
        ┃    ┃
        ┃ ┃  ┃
        ┃ ┃  ┃
        ┃ ┃  ┃
        ┃ ┃  ┃
   ─────┴──┴──┴─────────────
   ──·──·──·──  ← 短期均线（角度大）↗
   ──·──·──·──  ← 中期均线（角度中）
   ──·──·──·──  ← 长期均线（角度小）↗

   喇叭口特征：
   ① 短期均线角度大（涨幅大）
   ② 长期均线角度小（转变慢）
   ③ 各条均线形成喇叭口
   ④ 越往上走，喇叭口张得越大
   ⑤ 长期资金获利也很多

喇叭口张的大不大？这个是后话，但能不能提前知道，它以后的喇叭口会张得很大。后面的走势很多人都看到了，但是已经无法再参与了，那怎么样的口子才有会成为漂亮的喇叭口呢？这才是着重要观察的地方。其实我们之前就讲过很多次了，就是要在某个价格区间震荡的时间越久，各条长期均线、中期均线和短期均线反复粘合在一起。在这个过程中煎熬的时间越长，它后期的上涨幅度越大，在这个过程中我们可以看到股价呈现波浪式的上下波动，均线反复穿插，有可能是短期均线上穿长期均线，然后又下穿长期均线，这样反复缠绕在一起。他们就像在恋爱中的男女，偶尔低语呢喃，偶尔轻声笑语，有时尽情享受，有时如胶似漆，有时却貌合神离。在这个过程中纠葛的越多，后期双方就会更加珍惜彼此，这样的情况在股市中也是一样的，多条均线在底部纠缠的时间越长而上涨的空间也越大。

【底部均线纠缠——时间越长，空间越大】

       价格
        ┃
        ┃  ★底部均线纠缠
        ┃  ┃  ┃  ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃  ┃    ← 各条均线反复缠绕
        ┃  ┃  ┃  ┃  ┃  ┃      穿插在一起
        ┃  ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──·──  ← 短期均线
   ──·──·──·──·──·──·──  ← 中期均线
   ──·──·──·──·──·──·──  ← 长期均线
   （均线反复穿插缠绕）

   底部纠缠的意义：
   ① 短期均线上穿→下穿→再上穿长期均线
   ② 反复缠绕在一起
   ③ 纠缠时间越长 → 上涨空间越大
   ④ 就像恋爱中的男女：纠葛越多，越珍惜彼此
   → 底部盘整越久，后期涨幅越惊人 🚀

不知道大家还记不记得暴力吸筹的方法，主力资金在底部拿不到多少筹码，他们就通过一字板的方式，提高建仓的成本，通常拉了几个一字板之后，很多资金获利丰厚，他们喜滋滋的逃离了。他们不知道的是，主力资金通过这种方式吸收更多的筹码，他的目标远远不止是这几个一字板，在底部拿不到筹码，他们就通过这种方式拿到更多的筹码。不过一字板之后股价在某个特定的区间反复震荡，这个时间可能会很长，也可能不是特别长，短的就几天，长的可能需要几周。在这个位置震荡的目的就是为了吸收更多的筹码，同时甩掉这些跟风的筹码，因为这个过程中也有不少资金想赚更多的钱，但是不是所有的人都能忍受这样的煎熬。有的跟风资金在这个过程中亏损了，有的是这个过程中也是赚钱了，不知道后面会怎么走，怕赚到的钱可能也会亏回去，所以也可能会离场。其实让大多数的跟风盘离场就是主力资金的目的，在震荡的过程中可以发现出这样一个特点，一开始的抛压很大，量柱很高，随着反复震荡抛压越来越低。

【暴力吸筹——一字板后反复震荡】

阶段一：一字板拉升（提高建仓成本）
        ┃
        ┃  ★一字板  ★一字板  ★一字板
        ┃  ────  ┃  ────  ┃  ────  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃  ┃
   ─────┴──┴──┴──┴──┴──┴──┴──┴──┴─────────────
   （一字板拉升 → 底部拿不到筹码 → 提高成本吸筹）


阶段二：高位反复震荡（洗盘吸筹）
       价格
        ┃     ┃     ┃
        ┃  ┃  ┃  ┃  ┃  ┃
        ┃  ┃  ┃  ┃  ┃  ┃    ← 反复震荡
        ┃  ┃  ┃  ┃  ┃  ┃      短期均线反复缠绕
        ┃  ┃  ┃  ┃  ┃  ┃      长期均线由跌转涨
   ─────┴──┴──┴──┴──┴──┴─────────────
   ──·──·──·──·──·──·──  ← 短期均线反复缠绕
   ──·──·──·──·──·──·──  ← 长期均线（由跌转涨）

   震荡过程中的量能变化：
   一开始：抛压大，量柱高
   反复震荡后：抛压越来越低，量柱萎缩
   → 短期均线反复缠绕，成本趋于一致
   → 长期均线由跌转向上涨
   → 主力吸筹完成，拉升在即 🚀

在这个时候就需要重点观察了，因为在这个震荡阶段，短期均线也是反复缠绕的，这个时候短期资金他们的成本趋于一致。这个时候最重要的表现形式就是短期均线反复缠绕，长期均线由跌转向上涨，因为前期是通过暴力的一字板上涨的，长期均线转变的速度会比较慢一点，而且这个震荡期间，股价距离长期均线可能还会有些距离。
