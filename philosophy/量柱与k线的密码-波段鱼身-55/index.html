所谓的贪婪和恐惧的情绪，他们有的时候是很难把握住的，但是我们可以从k线的形态，或者一些均线的走向能辨别出来是哪种情绪，这也是咱们普通人能够介入的最好方式。之前我们也说过那些通过龙头战法的人，他们确实很牛逼，有些人确实能抓到真正的龙头，因为他们就是看中了里面贪婪的情绪的持续性。

龙头之所以会成为龙头，是因为里面的资金不断的愿意在里面接力，就是因为他们认定了这个是真正的龙头，它的某些特点就是具有龙头的潜质，所以有不少资金在那里不断的前赴后继。但这些还是有些比较难把握的，因为一旦贪婪过头了，随时就会变成恐惧，这种恐惧带来的伤痕不是一般人能承受得了的。有些股票看上去是像龙头，它连续有三四个板，里面贪婪的情绪也很浓厚，但是当某一天发现没有接力资金来的时候，这种贪婪瞬间就会变成恐惧。一只股票，它能成为龙头，它肯定具有某些特质，是其他股票所没有的，它能吸引源源不断的资金进来接手，从而造成贪婪情绪的强化。

【龙头战法的真相——贪婪瞬间变恐惧】

       贪婪情绪 😈
           ↓
     连续三四个涨停板
           ↓
     接力资金断裂 💔
           ↓
     贪婪 → 瞬间变恐惧 😱
           ↓
     大面积亏损 💸

   龙头战法的关键：
   ┌─────────────────────────────┐
   │ ① 需要源源不断的资金接力    │
   │ ② 一旦接力断裂→贪婪变恐惧  │
   │ ③ 需要强大心脏和自控力      │
   │ ④ 当天大亏+第二天可能跌停   │
   │ ⑤ 也可能第二天反包涨停      │
   │ ⑥ 充满不确定性              │
   └─────────────────────────────┘

作为普通的人是很难判断这种情绪是否会得到强化，很难判断第二天是不是有资金进场。因为当天进入的资金，他们判断今天会有更多资金进场，同时他们也判断第二天有更多的资金进来接手，这样他们才可以从容出局。但是这些判断都是自以为，第二天是不是能跟他们所希望的一样，那就不好说，只有龙头才能让他们从容出局。所以当天进入某一只所谓的龙头股票时，他们心中也是充满忐忑的，因为他们的愿望往往是很美好的，实际不见得和他们的愿望是一致的。

很有可能当天进去的时候他们就会大幅亏损，也有可能当天他们进入的时候也会涨停，他们能赚取丰厚的利润，如果当天就大幅亏损，第二天也有可能继续跌停，造成伤害更大。但是也有可能第二天能反包，如果他们忍受不了这个亏损的时候，当天开盘的时候就割肉出去，可能反包之后涨停，他们又会拍断大腿。在这个市场中就是这么刺激，当你充满希望的时候，就给你失望。当你失望感到绝望的时候，他又给你希望，这些情绪反复交织，把人折腾的不行。所以做龙头战法的时候，如果没有强大的心脏，没有那种保持自己清醒的能力，不能把控自己情绪的能力，在里面操作的那大概率是很痛苦的。

【龙头战法的情绪过山车】

  当天买入 ──┬── 涨停 ✅（赚钱）
             ├── 跌停 ❌（亏损）
             │
  第二天 ──┬── 继续跌停 ❌（大亏）
          ├── 反包涨停 ✅（赚钱）
          │
  割肉后 ──┬── 反包涨停 😭（拍断大腿）
          ├── 继续下跌 ✅（庆幸）

  结论：龙头战法情绪反复交织
  需要强大心脏 + 清醒头脑
  不适合绝大多数人 ❌

这也是我为什么说龙头战法并不适合绝大多数人的原因。尽管它很刺激，尽管它很赚钱，但是亏损也是往往不小。没有几个人有这样的自控能力，不要说龙头战法，就是在普通的情况下，他们也很难把握住自己的情绪，何况是那种刺激的大亏大涨的情况下，他们更加难控制好自己的情绪。如果一个人在普通的情况下，就是股票自然涨跌的情况下还能把握住，那么他可以去尝试一下，如果在普通涨跌的情况下都把握不住，压根不要去考虑龙头战法了，这种战法进去基本上都是做炮灰的。如果是想学那种龙头战法的，对不起，这里可能是不适合你，因为我也并不知道龙头战法是怎么来做，我只会脚踏实地的做自己能做的事情。

其实在实际的操作中能做到波段，那也是很厉害的，既然说到波段，那必然有底部或者是顶部，那怎么来判别底部和顶部呢？大家都知道波段是一个好东西，如果能抓到波段，那就是很厉害的事情，因为在这个过程中不需要追涨杀跌，只需要拿住就好了，可以享受超额的利润。这样可以节约很多时间成本，因为如果这样拿着，它不断下跌或者震荡，压根就没有任何利润，但是如果这种波段，在底部进入，上方都是赚钱，它可以节约时间成本。

【波段操作——脚踏实地】

       利润
        ┃        ★鱼身
        ┃       ╱  ┃  ╲
        ┃      ╱   ┃   ╲
        ┃     ╱    ┃    ╲
        ┃    ╱     ┃     ╲
        ┃   ★鱼头  ┃  ★鱼尾
   ─────┴───┴─────┴─────┴─────────────
   底部介入  持有  （赚钱）  顶部离场

   波段操作的好处：
   ┌─────────────────────────────┐
   │ ① 不需要追涨杀跌            │
   │ ② 只需要拿住就好            │
   │ ③ 享受超额利润              │
   │ ④ 节约时间成本              │
   │ ⑤ 底部进入→上方都是赚钱    │
   └─────────────────────────────┘

但是这样确实很难做得到的，因为在这个市场里面每天都有上涨，每天都有下跌，总是不想错过每一个上涨，但却不知道有的上涨就是在那里诱多，最终的结局还是下跌。这样会让人失去心智，会搞乱一个人的心态，频繁交易和过度交易，这样唯一的结局就是非理性，如果一个人带着情绪化的做任何事情，不可能能做得好，何况这是股市，更加不可能。

在震荡的市场里面，做波段是一个比较好的策略，因为如果在单边上涨或单边下跌的市场里面，只需要拿着或者空仓就可以了，当震荡市场里面局部还是有赚钱效应的时候，选取做一些有赚钱效应的股票，那也是不错的选择。那怎么样选择波段呢？这也是一个技术活。如果不好好拿捏，这是很难做好的，其实做好波段还是要把均线用好，也能很好的实现。其实做波段那也是一种短线操作，但与那种今天买明天卖的超短操作不一样，短线还是能够持续拿个3~5天，甚至更长的时间。依据这个逻辑，所以我们重点就看5日线和10日线，因为这两个数字就是代表短线的运行逻辑。

之前我们也说过，鱼头和鱼尾是没什么肉的，吃起来骨头特别多，也不是很舒服，要吃就吃鱼身，但是鱼身这一段该怎么判断呢？其实这个道理咱们之前很多人都已经说过了，当5日线上穿10日线的时候，那代表鱼头出现了，在这个时候股价处于10日线的上方就可以进入，这样介入的风险很低，挣钱的概率会高很多。当5日线上穿10日线的时候，其实就是鱼身阶段了，这个时候拿着并且持有它，那么同样的道理，5日线下穿10日线的时候，这个时候就需要离场了，这个时候该出现鱼尾了。

【波段鱼身的判断——5日线/10日线】

       价格
        ┃
        ┃              ★鱼身
        ┃             ╱  ┃  ╲
        ┃            ╱   ┃   ╲
        ┃           ╱    ┃    ╲
        ┃          ╱     ┃     ╲
        ┃   5日上穿10日  ┃  5日下穿10日
   ─────┴──────┴─────────┴──────┴─────────────
   ──·──·──·──·──·──·──  ← 5日线
   ──·──·──·──·──·──·──  ← 10日线

   波段鱼身的操作：
   ┌─────────────────────────────┐
   │ ① 5日线上穿10日线 = 鱼头出现│
   │ ② 股价在10日线上方 = 可以进入│
   │ ③ 介入风险低，赚钱概率高    │
   │ ④ 5日线下穿10日线 = 鱼尾出现│
   │ ⑤ 此时需要离场              │
   └─────────────────────────────┘

这个道理很多人都知道，也很明白，但是真正做得好的人却没有几个，当鱼头出现的时候介入，如果在出现鱼尾的时候，股价可能已经下跌了很多，或许在之前离开的话，可能挣的钱会更多。
