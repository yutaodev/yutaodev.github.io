笔记日期：2022-02-23
记录人：会写代码的小郎中，一个不喜欢麻烦、所以决定赚钱的人
心法来源：内心感悟

“7000万是我的目标。
但你得先做到：在赚到7000万之前，先学会做好每一笔交易，何时买入、卖出，控制好每笔交易的回撤。

背景

其实我这人不看中钱，因为我是一个重情义的人。

但后来生活中才发现：有钱真的能解决很多麻烦，而我这个人又不喜欢麻烦。

所以，决定在A股淘金，实现发财梦想。

我的炒股理念

一、炒股理念

  • 赌博你不一定能赢，但必须控制输。

  • 账户里的浮盈，仅仅是数字，只有落袋为安的利润才是真的。

  • 想从A股捞钱，必须投机，投机的本质是：在不确定的情况下，做出一个正确的决策。

二、肌肉记忆

股市最难的地方，是很难量化（影响股票的因素非常多，特别是一些细节），并且没有唯一答案（这次操作有效，下次就不灵了），吾之蜜饯，彼之砒霜。

我认为肌肉记忆是最好的风控，因为只有在严格的纪律下才能形成，任何一笔交易，做好两种操作：

  • 止盈线：根据情绪，在认为的高点，必须止盈
  • 止损线：根据情绪，在认为的低点，必须止损

不仅炒股，很多事练到别人以为你生来就会，练到你出手瞬间就是标准答案，所以肌肉记忆是完美的，无它。

游资交易

1. 北京炒家（六年8万到上亿）

1.1 简介

  • 风格定位：无情的打板机器人

  • 核心手法：首板模式（只打第一个涨停板）

  • 交易频率：极高，隔日交易为主，次日必卖

  • 资金容量：从单票满仓→双票满仓→四票→八票（超市防御流）

  • 核心理念：慢就是快，复利是世界第八大奇迹

1.2 核心交易特点

特点 详细说明
只做首板 不接二板、不打高位板、不做龙头接力
图形至上 优先选择K线光滑、分时控盘稳健的票
板块效应 秒拉板必须有板块联动，否则不打
分时细节 秒拉板要排板（不扫），换手板要扫板（不排）
回撤控制 设资金底线，跌破后猥琐发育，小仓位找感觉
牛市策略 快速满仓，不分热点贵贱，赚钱的都是钱

维度 分析
统计学特征 胜率约65-70%，单票盈亏比约1.5:1，仓位极度分散（8-10只）
行为学特征 极致纪律执行者，没有格局，第二天无论盈亏必走；回撤控制在3%以内时最舒服
对我最大的启发 ① 慢就是快，复利不需要单月翻倍；② 设底线不冒进，底线之上可大胆；③ 交易是为了更好的生活，不是赌博

2. 陈小群（30万到十亿）

2.1 简介

  • 风格定位：龙头信仰者，天才型选手

  • 核心手法：只做主流题材第一龙头，锁仓到涨不动

  • 交易频率：低频，但出手极其精准

  • 资金容量：从30万到10亿+（2024年）

  • 核心理念：情绪+合力+龙头

2.2 核心交易特点

特点 详细说明
只做主流 非主流不看，只做当下最强题材
龙头信仰 买入第一龙头后不断加仓，不做差价
反核按钮 龙头被核时敢于反核买入（特力A、中交地产经典案例）
情绪拐点 对情绪周期中拐点的把握极其精准
补涨逻辑 龙头首次分歧转强时，果断上后排补涨

3. 92科比（四年几万到2亿）

3.1 简介

  • 风格定位：全能型选手，手法最多样

  • 核心手法：龙头+补涨+切换，无定式

  • 交易频率：中等，分仓2-4个

  • 资金容量：从小资金分仓2个→大资金分仓4个

  • 核心理念：大道无形，情绪周期的四个阶段

3.2 核心交易特点

特点 详细说明
情绪周期 低位震荡（试错）→ 主升（大胆分歧买入）→ 高位震荡（轻仓）→ 主跌（尾盘找买点）
手法多样 打板、低吸、半路、竞价、尾盘，无所不能
题材预判 核心能力是判断一个题材能不能炒、能炒多大
仓位管理 从小资金分仓2个→大资金分仓4个，从不梭哈
回避高位 高位震荡阶段轻仓，多次分歧后的龙头不打板

特点 详细说明

阅读交割单后自我总结

一、我的交易三阶段模型

flowchart LR
    A[买点<br>北京炒家+缠论] --> B[持有<br>陈小群理解力]
    B --> C[卖点<br>心理价位+铁律]
    C --> D[闭环<br>稳定盈利]

二、买点：北京炒家首板模式 + 缠论底分型

融合点 具体做法
缠论底分型 底分型的“三条腿”分别是：左肩低点（第一腿）→ 最低点（第二腿）→ 右肩确认（第三腿）。最佳买点在第三腿刚形成、还没涨停时提前埋伏，或者等放量突破分型上沿时跟随。
北京炒家打板思维 不打无准备的板。即使是底分型确认后的首板，也要看：① 分时控盘是否稳健；② 是否有板块效应；③ 换手是否充分。

举个例子：某票日线走出标准底分型（三腿结构），次日分时温和放量，分时图在5%-7%区间横盘换手超过30分钟，这就是“换手板”的雏形——此时可以扫板或排板，确定性比纯缠论买点更高。

三、持有/卖点预判：学习陈小群的市场理解力

维度 思考问题
分时曲线 是急涨慢跌（出货）还是慢涨急跌（洗盘）？
市场情绪 当前是情绪主升、高位震荡、还是退潮？
人气/合力 这个票是龙头、跟风、还是独立庄股？
板块效应 板块内有没有梯队？龙头有没有超预期？

核心原则：买点可以机械化（底分型+首板），但持有和卖出不能用死规则。每天复盘问自己四个问题，答案就是次日操作的依据。

四、卖出操作：心理价位 + 铁律执行

类型 做法
设定心理价位 买入前就想好：① 止盈目标（如5%或10%）；② 止损底线（如-3%或-5%）；③ 时间底线（如次日不涨停就走）
坚定执行，不能有二心 这是最反人性但也最重要的一环。北京炒家次日必走，涅盘重升一字板也走，陈小群高位砸盘毫不犹豫——顶级选手的共同点不是买得准，而是卖得狠。
防止错失机会或亏损 不执行纪律 → 要么卖飞大腿拍断，要么死扛巨亏。执行纪律 → 可能卖飞，但长期看是正期望值。

五、整理自己的交易流程

flowchart TD
    START[交易机会] --> A[买点：北京炒家 + 缠论]

        A3[右侧跟随或排板<br>底分型上沿放量突破]

        B2[市场情绪<br>主升/高位震荡/退潮？]
        B3[人气与合力<br>龙头/跟风/庄股？]
        B4[板块效应<br>梯队完整？龙头超预期？]
    end

    C -->|持有信号| B
