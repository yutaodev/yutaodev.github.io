一、缠论的哲学根基

正闻、正见、正学、正行是缠论的哲学根基。

在进入具体技术前，缠师反复强调：市场不是赌场，是修行场。所有技术分析最终服务于“战胜自己的贪婪与恐惧”。

  • 不患与患：市场的风险是“不患”的（永远存在），但通过分类系统（走势分解、买卖点）可以将其转化为“患”（可操作、可应对）。
  • 当下性：走势是当下生长出来的，任何预测都是徒劳，唯一可靠的是根据当下走势的分类边界进行操作。
  • 零向量：把自己变成市场的零向量，不预设多空，只跟随合力方向。

二、基础构件

分型、笔、线段（形态学基石），这是递归定义最小级别走势类型的基础，必须严格掌握。

1. 分型

  • 顶分型：三根相邻K线，中间K线的高点是三根中最高的。
  • 底分型：三根相邻K线，中间K线的低点是三根中最低的。
  • 包含关系处理：相邻K线存在包含（一根K线的高低点完全在另一根范围内）时，向上趋势取“高高”（两K线高点取高、低点取较高者），向下趋势取“低低”（两K线低点取低、高点取较低者），合并为新K线。

2. 笔

  • 定义：相邻的顶分型和底分型之间，至少隔有一根独立K线（即顶与底之间至少有一根不属于分型的K线），构成一笔。
  • 方向：上升笔 = 底分型 + 上升K线 + 顶分型；下降笔 = 顶分型 + 下降K线 + 底分型。
  • 规则：顶和底不能共用K线；同一笔中顶必须高于底。

3. 线段

  • 定义：至少由三笔构成，且前三笔必须有重叠部分（形成最小中枢）。
  • 方向：由第一笔方向决定，向上线段以最低点开始、最高点结束；向下线段反之。
  • 线段破坏（结束）：必须被另一个反向线段破坏。分为两种情况：
    • 第一种（无缺口）：特征序列的顶/底分型中，第一、第二元素之间没有缺口，则线段在该分型高点/低点结束。
    • 第二种（有缺口）：第一、第二元素之间有缺口，则需要从该分型高点/低点开始的相反方向特征序列，再出现一个分型，才能确认线段结束。
  • 特征序列：向上线段的特征序列是向下笔的序列；向下线段的特征序列是向上笔的序列。处理包含关系后成为标准特征序列。

三、中枢与走势类型

1. 缠中说禅走势中枢

  • 定义：某级别走势类型中，被至少三个连续次级别走势类型的重叠部分所构成。
  • 最低级别：对于无法再分解的级别（如1分钟图），可用至少三个连续单位K线的重叠区域近似。
  • 中枢区间：设三个次级别走势的高低点分别为(a1,a2)、(b1,b2)、(c1,c2)，则中枢区间为 [max(a2,b2,c2) , min(a1,b1,c1)]。
  • 方向：中枢本身无方向，但产生方式分“回升中枢”（向上段开始）和“回调中枢”（向下段开始）。

1.1 先理解“次级别走势类型”

假设我们以 30分钟图 为观察级别（即本级别）。那么次级别就是 5分钟图（或更小）。

  • 一个5分钟走势类型：就是在5分钟图上，一个完整的上涨、下跌或盘整。它至少包含一个5分钟中枢，有明确的起点和终点。
  • 三个连续：指时间上前后衔接的三个5分钟走势类型，例如：上涨 → 下跌 → 上涨，或者 下跌 → 上涨 → 下跌。

1.2 什么是“重叠”？

“重叠”是指这三个次级别走势类型在价格区间上有共同的交集。

举个数字例子

  • 第一个5分钟上涨走势：价格从 10元 涨到 15元，它走过的价格区间是 [10, 15]。
  • 第二个5分钟下跌走势：从 15元 跌到 12元，区间是 [12, 15]。
  • 第三个5分钟上涨走势：从 12元 涨到 16元，区间是 [12, 16]。

三个区间的共同部分（重叠）是 [12, 15]。这个 [12, 15] 的区间，就是 一个30分钟级别的中枢。

1.4 用具体股票K线图理解

打开任意股票软件，把周期调到 5分钟图。

找一段走势：上涨一段 → 下跌一段 → 再上涨一段。然后用鼠标从最高点往下拉，看这三段在价格上重叠最多的那个水平区域。那个区域就是 15分钟或30分钟级别的一个中枢。

1.5 常见形态

  • 平台型中枢：价格在一个水平箱体内来回震荡（三个次级别走势来回穿越同一个价格带）。
  • 奔走型中枢：三个次级别走势虽然高低点不同，但仍有共同价格区间（例如第一个高点15，第二个低点13，第三个高点16，重叠区13-15）。

1.6 为什么必须是“三个连续”？

  • 两个次级别走势：只能形成一次“离开-返回”，无法确认多空平衡区间。
  • 三个次级别走势：最少需要三次重叠，才能形成有效的“成交密集区”，代表市场在这个价格区间内经过了充分的买卖较量。这个区域就是中枢，它是后续走势分析的“根据地”。

2. 走势类型

  • 盘整：任何级别中，只包含一个中枢的走势。
  • 趋势：任何级别中，包含两个及以上依次同向（无重叠）的中枢。
    • 上涨趋势：中枢依次抬高。
    • 下跌趋势：中枢依次降低。

2.1 盘整只包含一个中枢

是的，盘整就是整个走势中只有一个中枢。
这个中枢可以是任何形态（平台、奔走、三角形等），但只有一个。走势从这个中枢出发，离开它，然后可能再回来，但只要没有形成第二个同级别中枢，就依然是盘整。

2.2 两个及以上依次同向、无重叠的中枢

“依次同向”：中枢一个接一个，方向一致。上涨趋势中，每个中枢的高点、低点都比前一个中枢高；下跌趋势反之。 “无重叠”：两个中枢的波动区间（即中枢上下沿）没有任何交集，甚至不能有瞬间的触碰。

2.3 关键理解点

  1. 盘整只有一个中枢：无论震荡多久、延伸多少段，只要不形成第二个同级别中枢，都是盘整。
  2. 趋势必须至少两个中枢：一个中枢无法形成趋势。两个中枢必须是同向（都向上或都向下），且不重叠（包括波动区间也不重叠）。
  3. 为什么强调“无重叠”：如果两个中枢有重叠，就会融合成更大级别的一个中枢，那就又变成了盘整（或中枢扩展），不再是原级别的趋势。

3. 中枢的运动方式

  • 延伸：中枢形成后，后续次级别波动始终与中枢区间重叠，不产生第三类买卖点。延伸超过6段（加上形成中枢的3段，共9段）则升级为更大级别中枢。
  • 新生：产生新的同级别中枢且与前一中枢无重叠，形成趋势。
  • 扩张（扩展）：两个同级别中枢的波动区间产生重叠，从而形成更大级别中枢。

4. 走势必完美（第一原理）

  • 定理一：任何级别的任何走势类型终要完成。
  • 定理二：任何级别任何完成的走势类型，必然包含一个以上的缠中说禅走势中枢。
  • 核心意义：走势可以唯一分解为盘整、上涨、下跌的连接，从而可以动态把握其生、住、坏、灭。

四、买卖点

所有买卖点最终都可归结为某级别的第一类买卖点。

类型 位置 性质 实战意义
第一类买点（1B） 下跌趋势最后一个中枢之下，趋势背驰点 最安全 抄底最佳点
第二类买点（2B） 1B后的第一次次级别回跌结束点（不创新低） 高 加仓点
第三类买点（3B） 向上离开中枢后的第一次次级别回抽，不回到中枢内 较高 强势启动点
对应卖点（1S、2S、3S）反之
  • 重合情况：2B与3B可能重合（如V型反转），此时力度最强。
  • 注意：1B出现后，上涨级别至少是次级别。2B不必然出现在中枢之下，可在任何位置。

4.1 第一类买点（1B）

形成条件：一个下跌趋势，至少有两个同级别中枢（中枢A、中枢B），且第二个中枢之后出现趋势背驰（离开中枢B的下跌段力度比进入中枢B的下跌段弱）。

位置：最后一个中枢（中枢B）之下，背驰的最低点。

  • 中枢A、B依次降低（下跌趋势）
  • 中枢B后的下跌段创出新低，但其MACD柱面积/斜率等力度小于前面进入中枢B的那段下跌 → 背驰
  • 背驰的最低点 = 第一类买点
  • 此后必然至少反弹到中枢B的区间内

4.2 第二类买点（2B）

形成条件：第一类买点出现后，股价开始上涨（次级别走势向上）。然后必然有一次次级别回跌（不创新低），这个回跌的结束点就是第二类买点。

位置：可以在中枢下方、内部或上方，但一定高于第一类买点（不创新低）。

  • 1B后上涨一段，然后回跌
  • 回跌不破1B的低点，该回跌的终点 = 第二类买点
  • 安全性高，适合加仓

4.3 第三类买点（3B）

形成条件：股价向上离开最后一个中枢（可以是盘整的中枢，或下跌趋势最后一个中枢被突破后形成的新中枢），然后第一次次级别回抽，回抽的低点不回到中枢区间内。

位置：中枢之上，且回抽低点高于中枢的上沿（ZG）。

  • 向上离开中枢的段可以是次级别走势（或缺口）
  • 之后第一次次级别回抽，其低点高于中枢上沿（ZG） → 第三类买点
  • 之后往往有快速上涨（或形成更大级别盘整）

4.4 卖点（1S、2S、3S）

把上述图形上下翻转，就是对应卖点：

  • 第一类卖点（1S）：上涨趋势最后一个中枢之上，顶背驰的最高点。
  • 第二类卖点（2S）：1S后的第一次次级别反弹结束点（不创新高）。
  • 第三类卖点（3S）：向下离开中枢后的第一次次级别回抽，回抽高点不回到中枢内。

4.5 三类买点总结对比

买点 位置 条件 后续最低保证
1B 最后一个中枢之下 下跌趋势背驰 回到最后一个中枢区间
2B 任何位置（不低于1B） 1B后第一次次级别回跌结束 至少再有一段次级别上涨
3B 中枢之上 向上离开中枢后，次级别回抽不返回中枢 中枢新生或扩展，大概率强势上涨

五、背驰

背驰是判断走势转折的关键，但必须先有趋势，才有背驰（盘整中类似判断叫“盘整背驰”）。

1. 趋势背驰（标准背驰）

定义：在一个下跌趋势（至少两个同级别中枢A、B）中，最后一个中枢B之后的向下离开段c，其下跌力度小于进入中枢B的向下段b，称为底背驰（上涨趋势反之称为顶背驰）。

结构：a（第一段）+ A（中枢1）+ b（第二段）+ B（中枢2）+ c（第三段）

下跌趋势底背驰

关键观察：

  • c段创出了新低（低于b段的最低点）
  • 但c段对应的MACD红柱子面积（或黄白线高度）比b段小
  • 价格新低，MACD不新低 → 底背驰
  • 上涨趋势底背驰同理，价格新高而MACD红柱子面积缩小

2. 盘整背驰

  • 结构：在一个盘整中，离开中枢的段（如C段）与进入中枢的段（如A段）比较力度。
  • 结果：不一定回中枢，但会引发次级别回调/反弹；若回试不破中枢，可能形成第三类买卖点。

3. MACD辅助判断

  • 黄白线至少回拉0轴一次。
  • 比较相邻同向段的红绿柱面积（或高度），后段面积小于前段为背驰信号。
  • 注意：MACD只是辅助，最终以走势结构为准。

4. 背驰-转折定理

某级别趋势的背驰将导致三种可能：

  1. 该趋势最后一个中枢的级别扩展；
  2. 该级别更大级别的盘整；
  3. 该级别以上级别的反趋势。

5. 一个完整的趋势背驰实例

假设一段30分钟上涨趋势：

价格：  10-12区间   14-16区间    创新高到18

  • 再涨到16，回调到14形成B中枢[14,16]
  • 然后涨到18（创新高），但MACD柱子面积比从A到B那段（b段）小很多
  • 背驰确认！后续必然至少回到B中枢区间[14,16]内

六、级别

  • 定义：走势按时间周期或递归定义的层次。不同级别如同不同倍数的显微镜。
  • 递归：最小级别（如1分钟图）的线段构成1分钟中枢 → 1分钟走势类型 → 5分钟中枢 → 5分钟走势类型 → ……。
  • 操作级别：根据资金量、交易时间、技术水平等选择一个固定级别进行操作。在该级别上，不参与任何调整或下跌。
  • 区间套：通过逐级缩小级别，精确定位大级别转折点（例如从日线背驰段 → 30分钟图 → 5分钟图 → 1分钟图找到精确买点）。

七、走势分解与组合

1. 同级别分解

  • 将走势按固定级别分解成一段段该级别“盘整、上涨、下跌”的连接。
  • 特点：允许“盘整+盘整”；不需考虑中枢延伸、扩展；分解唯一。
  • 操作：只参与上涨段，下跌段空仓或做短差。

2. 非同级别分解

  • 利用结合律，对走势进行不同的组合，以便于当下判断。
  • 原则：保证中枢的确立，组合应有利于识别买卖点。
  • 中阴阶段：前一个走势类型完成、新走势类型未确立前的模糊阶段，表现为中枢震荡，可用中枢震荡方法应对。

3. 走势分解定理

  • 定理一：任何级别的任何走势，都可分解成同级别“盘整、下跌、上涨”的连接。
  • 定理二：任何级别的任何走势类型，都至少由三段以上次级别走势构成。

八、市场与心态

缠论不只是技术，更是心法。

  • 杜绝喜好：股票只是废纸，能涨的就是好股票，不要有感情。
  • 只搞能搞的：通过分类系统（技术面+比价关系+基本面）选出能搞的股票，然后严格按照买卖点操作。
  • 成本为0：通过中枢震荡、短差操作，将持仓成本降为0，然后不断增加筹码，直至大级别卖点清仓。
  • 零向量：不预测、不幻想，当下观察走势，根据分类边界反应。
  • 钢铁战士：买点总在恐慌中形成，卖点总在疯狂中形成；手起刀落，绝不拖泥带水。

九、实战操作流程（从看盘到下单）

步骤1：确定操作级别

例如选定30分钟级别。

步骤2：打开走势图，找到当下之前最后一个该级别中枢

判断当前股价位于中枢之上、之中、还是之下。

步骤3：根据位置，预判可能的买卖点

  • 在中枢之下：等待底背驰（1B）或盘整背驰（2B）。
  • 在中枢之中：可做次级别短差，等待突破。
  • 在中枢之上：等待回调不破中枢（3B），或顶背驰卖出。

步骤4：用区间套精确定位

  • 在背驰段内，切换到次级别图，找次级别的第一类买卖点。
  • 可一直追踪到1分钟甚至分笔图。

步骤5：买入后，持有直到卖点出现

  • 若走势出现第三类买点，继续持有。
  • 若出现盘整背驰或顶背驰，减仓或清仓。

步骤6：资金管理

  • 永远不用压力资金，不借贷。
  • 每只股票一次性买入，不加码。
  • 用机动资金（如1/10仓位）做短差，降低持仓成本。
