第6章 高量柱

出现高量柱的时候，往往是大量的人卖出，同时大量的人买进。好像很多人去买，一般的散户很难看清楚，这是主力进货还是主力出货？高量柱就有很难辨别其特征，就是搞不懂大资金的真实意图，当以为是大资金入驻时，却是出货，当以为是出货时，却在收集筹码。自然而然的，结果也完全不一样，如果是出货，随后会大跌。如果是收集筹码，价格能稳住，还可能大涨。高量柱是别人很难判别其主要用意的量柱，很多人卖，也很多人买，有情绪比较高昂的特征。不过这么大的量，肯定是大资金介入了，只不过介入的意图还不是很明确。它是很多资金在相互搏杀，不光是大资金跟小散户，还有大庄跟小庄，明庄跟暗庄，强庄跟弱庄。它的表现形式要不是很惨烈，要么是轰轰烈烈不断上涨。它的后续走势很难预测，此时空方很猛，虽然看上去多方也很厉害，但究竟是往哪个方向走？还不能确定。只是当日达到一个暂时的平衡。在一般情况下，出现那么多量，卖家肯定是优先原则，通常情况股价是下跌，但是非常态的情况下，它也会上涨。作为普通的投资者，只能是跟随，如果是非正常状态上涨，跟随就好了。

📊 形态定义：高量柱的本质——当日多空平衡，方向待定。

【高量柱的本质——当日平衡，方向待定】

        高量柱出现
            ┃
            ┃         多方 💪
            ┃    ┃    空方 💪
            ┃    ┃    （情绪都很高昂）
            ┃    ┃
   ─────────┴────┴─────────
           当日多空平衡
   ┌─────────────────────┐
   │  卖家多 ← 平衡 → 买家多  │
   │  空方猛 ← 平衡 → 多方猛  │
   └─────────────────────┘
   └──────────┬──────────┘
         方向待定 ❓
    ┌──────┴──────┐
    ↓              ↓
  常态下跌       变态上涨
  （卖家优先）   （该跌不跌）

不过出现高量柱的时候，多方和空方情绪都很上头，不但表现为卖方很强势，同时买方也很强势，特别是在轰轰烈烈上涨的时候。这个时候很容易勾起外面资金的情绪，让人会很激动，也很容易让人上头。很多人认为高量柱是大资金介入的表现，随后必有一波上涨。也有不少人认为高量柱是资金撤退的表现，随后必有一波下跌。其实这两种说法都不完全准确，在有的地方高量柱时介入，很有可能被套，而在有的位置，高量柱时介入，随后又会有一大波上涨。上涨会让以为出现高量柱时，是大资金撤退表现的人拍断大腿，出现高量柱时撤退的说法也是不完全正确的。但在绝大部分情况下，出现高量柱的情况是需要撤退的，如果盲目介入，大概率是会被套。

但是有时出现高量柱时，紧接着缩量下跌，如果再放量上涨，突破高量柱对应的k线实体价格时，又会迎来一波上涨。当出现一根高量柱时，通常会有两种情况值得密切关注，一个是变态的缩量上涨，另外一个是缩量下跌后，却放量上涨。当出现这两种情况时，意味着后续还会上涨，这也就是咱们发现高量柱时，需要密切关注的点。假如出现高量柱时，常态下跌，萎靡不振，这是不需要关注的。当出现高量柱时，通常会让人上头的发烧的，这种情况要尽量避免，通常它是撤退的信号，随后会下跌。当出现缩量上涨时，或许它是上涨行情刚刚启动，此时应该密切观察，在适当的时候考虑介入。

📊 形态定义：高量柱后的三种走势——常态下跌、变态一（缩量上涨）、变态二（缩量跌后放量涨）。

【高量柱后的三种走势对比】

走势一：常态下跌（❌ 不需要关注）
        ┃
        ┃  ★ 高量柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃     ← 价格萎靡不振
        ┃  ┃  ┃  ┃       持续下跌
   ─────┴──┴──┴──┴───
   高量柱  后续下跌
   → 撤退信号，不关注 ❌


走势二：变态一——缩量上涨（✅ 密切关注）
        ┃     ┃     ┃
        ┃  ★ 高量柱  ┃
        ┃  ────  ┃  ┃     ← 量柱缩小
        ┃  ┃  ┃  ┃  ┃       价格却上涨
   ─────┴──┴──┴──┴──┴───
   高量柱  缩量上涨
   → 上涨行情刚启动 📈


走势三：变态二——缩量跌后放量涨（✅ 密切关注）
        ┃            ┃
        ┃  ★ 高量柱   ┃
        ┃  ────  ┃   ┃     ← 缩量下跌后
        ┃  ┃  ┃  ┃   ┃       放量上涨
   ─────┴──┴──┴──┴───┴───
   高量柱 缩量跌 放量涨
   → 突破高量柱实体价格时
   → 迎来一波上涨 📈

当出现一根上涨k线和放量柱，后续两三根k线都是缩量的，价格却不停的上涨，这就是明显的价升量缩，应该立马介入。当出现连续几个高量柱的时候，这个时候就需要谨慎了，特别是出现高量柱价格却不涨反跌的时候。很有可能大资金在悄悄的撤退，仅仅酝酿了很热闹的情绪而已，特别出现多倍的量柱时，更加需要警惕。

量柱温和增加，价格也温和上涨，在这个时候量柱需要高看一眼，因为它有一点点狂热，但是有克制的，或者说是不动声色的，遮遮掩掩的。这个时候很有可能是大资金在背后操控着，不想让过多的资金关注。当出现价升量缩时，它的操控逻辑更加明显了，就是想拉升股价。根据卖在买先的原则，为什么会出现这样量缩？是因为有人不想卖而已。为什么会不想卖呢？最主要价格没达到心理的预期，只能不停的上涨，通过上涨来制造出卖出的欲望。可想而知，出现这种情况的时候，只有上涨才能出来量，而这也是介入的最好时机。

我们关注的更多是高量柱，更多的是关注出现这些量柱之后的k线表现。价格不跌反涨并且量柱不断的降低，就是好的量柱组合。如果出现高量柱或者倍量柱，但价格不断下跌，并且跌破了倍量柱的实体价格，那么它一文不值。我们就是要关注高量柱的反常态现象，该跌不跌反而上涨的情况。这种非正常的状态一般情况下，比较少出现，当出现这种情况的时候，就需要密切关注了。如果出现了很多这种形态股票的时候，反而就需要小心了。这种情况本来就比较少发生，如果大面积出现了，有可能是市场将发生大级别的上涨。

📊 形态定义：价值连城 vs 一文不值——同样的高量柱，后续表现不同，价值天差地别。

【价值连城 vs 一文不值】

✅ 价值连城（反常态——该跌不跌）：
        ┃     ┃     ┃
        ┃  ★ 高量柱  ┃
        ┃  ────  ┃  ┃  ← 价格不跌反涨
        ┃  ┃  ┃  ┃  ┃    量柱不断降低
   ─────┴──┴──┴──┴──┴───
   高量柱  后续表现
   → 好的量柱组合 🌟
   → 密切关注！


❌ 一文不值（常态——该涨不涨）：
        ┃
        ┃  ★ 高量柱
        ┃  ────  
        ┃  ┃  ┃  ┃  ← 价格不断下跌
        ┃  ┃  ┃  ┃    跌破实体价格
   ─────┴──┴──┴──┴───
   高量柱  后续下跌
   → 一文不值 ❌

当出现一根高量柱的时候，紧接着又出现了一根缩量柱，并且缩量柱比前一日缩量一半以上，同时价格未能跌破前一日实体的1/2位。说明什么问题？大家都不愿意卖出，大家都很珍惜筹码，为什么不卖出？这个价位大家都觉得适中，不值得卖出。这样会导致大家都不想砸，如果有少许资金点火，股价就会上涨。或许出现一根十字星，如果带长上引线的十字星更好，意味着大资金在试探上方的抛压。如果试探上方的抛压不重，也呈缩量状态，有可能即将发生大涨。道理也是很容易理解的。一般出现高量柱的时候，就意味着股价会下跌。如果一反常态，该跌不跌的时候，就是我们需要关注的时候，出现高量柱，一般情况下是大家都在卖出，如果不跌反而呈缩量状态，意味着虽然下跌，但还是有人不愿卖出筹码。也意味着有人对后续走势相当看好，所谓的顺势就是这样的，当出现高量柱的时候，价格一直下跌，意味着大家对此价位不看好。此根高量柱的价格就成为后续的压力线，如果后续股价突破了这个压力位，后续也会看涨。

📊 形态定义：高量柱后缩量柱 + 价格未跌破1/2位 = 大家惜售。

【高量柱 + 缩量柱 + 不破1/2位】

高量柱后出现缩量柱（缩量一半以上）：
        ┃
        ┃  ★ 高量柱
        ┃  ────  ┃
        ┃  ┃  ┃  ┃     ← 缩量柱
        ┃  ┃  ┃  ┃       价格未跌破前日实体1/2位
   ─────┴──┴──┴──┴───
   高量柱 缩量柱
   └──────────┬──────────┘
   缩量一半以上 + 价格不破1/2位
   → 大家都不愿意卖出 📦
   → 珍惜筹码，惜售明显
   → 少许资金点火即涨 📈

当出现高量柱的时候，先不用着急介入，先观察后续的价格及量柱状态。当价格突破高量柱对应k线实顶时，同时量能也不是放得很大，这个时候就是介入的好时机。不过同时需要注意的是，突破之后它必定有回踩，回踩不会跌破前一根高量柱实顶线，代表是真实的突破。在这种情况下可以放心持有，在这样的k线和量柱组合下，后续看涨。高量柱的k线和没有跌破实顶价格的k线组合有多种运用，也不需要太多的战法，只要抓住它的精髓就可以了。
