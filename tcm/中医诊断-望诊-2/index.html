望诊，是指医生通过视觉对人体的全身、局部及排出物等方面进行有目的的观察，以了解健康状况，测知病情的方法。望诊作为四诊之首，在诊法中必然占有极其重要的地位，故《难经·六十一难》云:“望而知之谓之神。”本博文是 《中医诊断》 博文系列的 第二篇，若想详细了解请点击 首篇博文。

1. 望全身

全身望诊，又称整体望诊，指医生通过对患者的神气色泽形体及姿态等进行整体观察，借以了解机体精气的盛衰、脏腑功能的强弱，作为辨别疾病性质、推断病情预后的依据。

1. 望神

2. 望色

望色，即观察人体皮肤色泽的变化来诊察病情的方法，也称“色诊”。色诊考察两个方面：

  • 色：指皮肤的颜色（色调），主要观察青、赤、黄、白、黑五种颜色；
  • 泽：指皮肤的光泽（明亮度），是脏腑精气盛衰的反映。

色诊基本原理与要素

  1. 面色原理：

    • “血气上荣于面”，即脏腑气血充足则面色红润，反之则失色。

  2. 色诊要素：

    • 颜色：主要观察青、赤、黄、白、黑五种变化；
    • 光泽：反映脏腑精气的充盈或衰竭，是“脏腑精气盛衰之镜”。

颜色的分类及其变化

  1. 颜色三态：

    • 常色：
      • 主色：先天禀赋决定的正常面色；
      • 客色：受四时更替等外在因素影响而呈现的颜色变化。
    • 病色：
      • 善色：病中出现的吉祥颜色；
      • 恶色：反映真实脏器病变的异常颜色；
      • 五色主病：青、赤、黄、白、黑各自对应特定病理状态。

  2. 望色十法：
    通过以下方法来分析面色，帮助辨别病情：

    • 浮沉分表里：观察面色是否浮现于表（表证）或沉隐于里（里证）；
    • 清浊审阴阳：通过色彩的清浊程度判断阴阳平衡；
    • 微甚别虚实：细致观察面色的细微变化，区分虚证与实证；
    • 散抟辨新久：区分病势是散漫还是聚集，新旧病证；
    • 泽夭测预后：根据皮肤光泽判断病情的严重程度和预后。

五色主病解析

  1. 青色（寒凝、气滞、血瘀）：

    • 面色淡青：提示寒滞于肝脉；
    • 青灰唇紫：提示心阳衰微；
    • 青黄相间：提示肝郁脾虚；
    • 眉间泛青：提示小儿惊风。

  2. 赤色（实热或虚热）：

    • 满面通红：提示阳明经热盛；
    • 颧红如妆：提示虚阳浮越；
    • 潮红盗汗：提示阴虚火旺。

  3. 黄色（脾虚湿蕴）：

    • 萎黄无华：提示气血两虚；
    • 黄胖虚浮：提示土虚湿困；
    • 身目俱黄：提示湿热或寒湿。

  4. 白色（气血暴脱）：

    • 淡白无华：提示营血亏虚；
    • 㿠白虚浮：提示阳虚水泛；
    • 苍白厥逆：提示气随血脱。

  5. 黑色（肾虚血瘀）：

    • 黧黑晦暗：提示命门火衰；
    • 焦枯如炭：提示肾水枯涸；
    • 紫黑甲错：提示久病入络；
    • 眶周晦暗：提示寒湿带下。

3. 望形

望形，是指通过观察患者形体的强弱、胖瘦及体型特点来诊查病情的方法，又称望形体。

望形原理

  • 《灵枢》记载：“皮肉脉筋骨，五体应五脏”。
  • 形体观察主要关注以下要点：
    • 皮：看皮肤的荣枯状态
    • 肉：观察肌肉的坚软情况
    • 脉：评估脉象是否充盈
    • 筋：考察筋的刚柔程度
    • 骨：检测骨骼的强弱情况

望形三纲
望形分为三个方面：形体强弱、形体胖瘦和形体体质。

  • 形体强弱

    • 体强特征（气血旺盛）
      • 骨骼健壮
      • 胸廓宽厚
      • 肌肉充实
      • 皮肤润泽
    • 体弱特征（气血不足）
      • 骨骼细小
      • 胸廓狭窄
      • 肌肉消瘦
      • 皮肤干枯

  • 形体胖瘦

    • 肥胖辨证
      • 如果能食，说明形气有余
      • 如果食量少，则可能是脾失健运
    • 消瘦辨证
      • 多食但依然消瘦，提示中焦火炽
      • 少食则可能是中气虚弱

  • 形体体质

    • 阴脏人（表现为阳虚阴盛）
      • 特征：体胖颈短，喜热恶凉
    • 阳脏人（表现为阴虚阳盛）
      • 特征：体瘦颈长，喜凉恶热
    • 平脏人（阴阳平衡）
      • 特征：体型适中，寒热适宜

4. 望态

望态，又称望姿态，是指观察患者的动静姿态和肢体异常动作以诊察病情的方法。望态是望诊的一部分，其基本理论源自《内经》：“阳主动，阴主静”，即动静姿态反映着阴阳的消长。诊断时需结合动静、形神和病证，全面考察。望态分为三大方面：动静姿态、动作异常和衰惫姿态。

望态原理

  • 基本理论：
    《内经》云：“阳主动，阴主静”，说明人体的动静状态反映着阴阳的平衡与消长。
  • 诊断要点：
    • 动静合参
    • 形神兼察
    • 病证结合

望态三纲

  1. 动静姿态

    • 坐姿：
      • 仰首挺胸 → 表示实证或热证
      • 蜷缩俯首 → 表示虚证或寒证
    • 卧式：
      • 面壁向里 → 指阴证、虚证
      • 仰面向外 → 指阳证、实证
    • 立姿：
      • 行立不稳 → 提示肝肾亏虚
      • 久立不动 → 可能是痰湿阻滞
    • 动态：
      • 步态蹒跚 → 见于风寒湿痹
      • 震颤摇摆 → 反映肝风内动

  2. 动作异常

    • 热证动风
      • 特征：四肢抽搐、角弓反张
    • 阴虚风动
      • 特征：手足蠕动、肌肉瞤动
    • 虚证失养
      • 特征：手足拘急、动作迟缓
    • 危重证候
      • 特征：如“循衣摸床”与“摄空理线”，提示病情危重

  3. 衰惫姿态

    • 坐姿衰惫
      • 坐而仰首 → 指肺气壅滞
      • 坐而俯首 → 指宗气不足
    • 立姿衰惫
      • 站立不稳 → 指肾精亏虚
      • 扶物而立 → 指阳气虚衰
    • 特殊姿态
      • 猝倒昏仆 → 痫证特征
      • 强迫体位 → 痛证标志

经方讲解：

  • 黄芪建中汤，治疗虚劳，症见气短乏力，自汗怕风。
  • 桂枝甘草汤，治疗心悸，症见心慌，欲手按护，寒象。

2. 望五官

目、耳、口、鼻、舌五官，分别与五脏相关联。《灵枢·五阅五使》云:“鼻者肺之官也，目者肝之官也，口唇者脾之官也，舌者心之官也，耳者肾之官也。”因此，观察五官的神、色、形、态变化，可以了解相关脏腑的常与变。其中，望舌将另做专节论述，故本处主要介绍目、耳、、口唇、齿龈及咽喉等望诊内容。

望目形
观察目形时，主要考察三个方面：

  • 胞睑肿胀
  • 眼窝凹陷
  • 眼球突出

胞睑肿胀

  • 水肿病初起
    • 表现：目胞浮肿，形似新醒时的浮肿状态，皮肤颜色不变且依然光亮。
  • 针眼
    • 表现：眼睑边缘出现麦粒肿样小肿块，伴有红肿、痒痛，易化脓。
    • 解释：提示风热毒邪相搏。
  • 眼丹
    • 表现：全睑漫肿，红得如涂丹，伴有明显热感，如火灼般可能化脓。
    • 解释：提示脾胃蕴积热毒。

眼窝凹陷

  • 吐泻伤津
    • 表现：因吐泻后，目眶出现凹陷。
  • 精气衰竭
    • 表现：长期或重病时，眼部凹陷且失神，反映精气衰竭。

眼球突出

  • 肺胀
    • 表现：眼球突出，伴有气喘；同时，可伴有痰浊阻肺。
  • 瘿病
    • 表现：眼球突出同时伴有颈部肿胀。
    • 解释：提示肝郁化火，痰结所致。

望目态

瞳孔缩小

  • 可能原因：
    • 肝胆火炽
    • 劳损肝肾/虚火上扰
    • 中毒：如吗啡、川乌、草乌
    • 毒蕈、有机磷农药中毒

瞳孔散大

  • 可能原因：
    • 肾精耗竭（濒危征象）
    • 肝胆风火（例如绿风内障）
    • 中毒：如杏仁、麻黄、曼陀罗
    • 药物性中毒：如阿托品等

目睛凝视

  • 表现形式：
    • 瞪目直视（前视）
    • 戴眼反折（上视）
    • 横目斜视（侧视）
  • 可能原因：
    • 肝风内动（伴有神昏抽搐）
    • 精气耗竭/痰热内闭
    • 瞪目直视有时可见于瘿病

嗜睡露睛

  • 可能原因：
    • 脾虚清阳不升
    • 津液大伤（如吐泻患儿）
    • 生理性（如“羊眼”现象）

睑下垂

  • 可能原因：
    • 先天性（通常双眼下垂）
      • 可能与脾肾亏虚或睑肌失养有关
    • 后天性（通常单眼下垂）
      • 可能与脾气虚衰、脉络失养或外伤性因素有关

望耳

主要包括三个方面：耳之色泽、耳之形态、以及耳内病变。同时，耳与脏腑之间存在密切关系（耳为肾之窍、宗脉所聚，关联肾、胆及全身）。

耳之色泽

润枯状况

  • 正常红润：提示气血充足。
  • 焦黑干枯：提示肾精亏虚。

颜色变化

  • 淡白：提示气血亏虚。
  • 红肿：提示肝胆湿热或热毒上攻。
  • 青黑：提示阴寒内盛或剧痛。
  • 小儿耳背红络：为麻疹先兆。

耳之形态

耳廓形大

  • 厚大：提示肾气充足。
  • 肿大色红：提示少阳相火上攻。

耳廓瘦小

  • 瘦薄：提示先天亏损。
  • 瘦削干焦：提示肾精耗竭或肾阴不足。
  • 萎缩：提示肾气竭绝。

耳轮甲错

  • 肌肤甲错：提示久病引起血瘀。

耳内病变

耳内流脓（脓耳）

  • 实证型：表现为脓黄稠且剧痛，提示风热上扰或肝胆湿热。
  • 虚证型：表现为脓清稀且痛缓，提示肾阴虚或虚火上炎。

耳道红肿

  • 若呈椒目状，则为耳疖，提示邪热搏结于耳窍。

望鼻

主要包括三个方面：鼻之色泽、鼻之形态和鼻内病变。同时，要注意鼻与脏腑的关系：鼻为肺之窍，也属脾，同时通于足阳明胃经。

鼻之色泽

  1. 润枯情况

    • 微黄明润：见于新病，提示胃气尚未受损。
    • 久病时：可能出现胃气来复的迹象。
    • 晦暗枯槁：提示胃气已衰。

  2. 颜色变化

    • 白色：提示气血亏虚。
    • 赤色：提示肺脾蕴热。
    • 黄色：提示湿热。
    • 青色：提示阴寒，常伴有腹痛。
    • 小儿山根青筋：提示肝经气滞或积滞。

鼻之形态

  1. 鼻头肿胀

    • 红肿生疮：提示胃热或血热。
    • 暗红血络：见于酒齄鼻，反映肺胃蕴热和血瘀。

  2. 鼻柱溃陷

    • 梅毒：可引起鼻柱溃陷。
    • 麻风：常导致鼻柱塌陷，并伴有眉毛脱落。

  3. 鼻翼扇动

    • 见于肺热或哮病，可能导致呼吸困难；
    • 在危重情况下，表现为喘息和额部多汗，如油般湿润。

鼻内病变

  1. 鼻流清涕

    • 见于风寒表证，常伴有寒热症状；
    • 鼻鼽现象提示阳虚，流涕量较多。

  2. 鼻流浊涕

    • 见于风热表证，常伴有咽痛；
    • 鼻渊则提示胆经蕴热，伴有腥臭气味。

  3. 鼻腔出血（鼻衄）

    • 外感：风热燥邪可引起鼻出血；
    • 实证：提示肝火或胃火亢盛；
    • 虚证：可能因脾不统血；
    • 倒经：提示肝郁或阴虚。

  4. 鼻内赘生物

    • 如鼻痔（息肉）：提示湿热壅结，导致气息难通。

望口唇

中医诊断的重要方法，通过观察口部和唇部的变化，可以判断脏腑功能、气血津液的盛衰以及病证的寒热虚实。口唇与脾胃、心、肺等脏腑密切相关，尤其是脾开窍于口，手足阳明经（胃经和大肠经）环绕口部。

望口

口角流涎

  • 小儿：脾虚湿盛，称为“滞颐”。
  • 成人：中风导致口角歪斜。

口疮

  • 原因：心脾积热或阴虚火旺。
  • 反复发作：中气不足。

鹅口疮

  • 原因：心脾积热或肾阴虚火。

口之动态

  • 口张：肺气将绝。
  • 口噤：痉病或中风。
  • 口撮：破伤风或脐风。
  • 口僻：风邪中络。
  • 口振：疟疾初起。
  • 口动：胃气虚或动风。

望唇

唇色

  • 淡白：血虚或失血。
  • 深红：热盛。
  • 青紫：阳虚血瘀。
  • 青黑：寒凝血瘀。
  • 干裂：津液损伤。
  • 糜烂：脾胃积热。

唇态

  • 干裂：燥热伤津。
  • 糜烂：脾胃积热或虚火。
  • 生疮：心脾积热。

望齿龈

判断肾、胃、脾等脏腑的功能状态，以及气血津液的盛衰和病证的寒热虚实。肾为牙齿为“骨之余”，由肾主骨，牙齿坚固与肾气密切相关。胃肠，牙龈属手足阳明经（胃经和大肠经），其色泽和形态反映胃肠功能。温病验齿，观察牙齿可了解胃津和肾液的盛衰情况。

望牙齿

牙齿形色

  • 正常：牙齿润泽，反映肾气充足。
  • 干燥：牙齿无光泽，提示胃阴已伤。
  • 光燥如石：牙齿干硬如石头，表示阳明热甚。
  • 燥如枯骨：牙齿干枯似枯骨，说明肾阴枯竭。
  • 枯黄脱落：牙齿发黄并脱落，称为“骨绝”，预后不良。

牙齿动态

  • 牙关紧急：牙关紧闭，可能由风痰阻络或热极动风引起。
  • 咬牙啮齿：牙齿咬合或摩擦，常因热盛动风或痉病导致。
  • 睡中啮齿：睡眠时磨牙，可能与胃热、食滞或虫积有关。

望牙龈

牙龈色泽

  • 淡红：牙龈呈淡红色，表示胃气充足。
  • 淡白：牙龈苍白，提示血虚或失血。
  • 红肿：牙龈红且肿胀，反映胃火亢盛。

牙龈形态

  • 牙宣（萎缩/渗血）：牙龈萎缩或渗血，多因肾虚或胃阴虚。
  • 牙疳（溃烂腐臭）：牙龈溃烂且有腐臭味，表明胃腑积热并伴疫毒。

齿衄（牙缝出血）

  • 原因：牙缝出血，可能由以下三种情况引起：
    • 胃肠实热。
    • 虚火上炎。
    • 脾不统血。

望咽喉

原理

  • 肺胃：咽喉是肺胃之门户，肺胃热毒常导致红肿、溃烂或伪膜。
  • 肾：肾经循行经过咽喉，肾阴虚火可引起咽喉嫩红微肿或虚火上炎的溃烂。
  • 疫毒：疫毒侵袭咽喉时，易形成坚韧白喉，需特别注意其特殊性。

红肿

  • 实热证
    • 表现：咽喉深红肿痛。
    • 病因：风热或肺胃热毒。
  • 阴虚证
    • 表现：咽喉嫩红微肿。
    • 病因：肾阴虚火。
  • 痰湿凝聚
    • 表现：咽喉淡红漫肿。
    • 病因：痰湿内阻。
  • 乳蛾（喉核红肿）
    • 急性：咽喉红肿，伴喉核明显，多由风热或肺胃热盛引起。
    • 慢性：红肿反复，多由肺肾阴虚导致。
  • 喉痛（红肿高突）
    • 表现：咽喉红肿明显隆起，疼痛剧烈。
    • 病因：脏腑蕴热加外邪侵袭。

溃烂

  • 新病溃烂
    • 表浅红边：溃烂较浅，边缘红，提示肺胃热轻。
    • 成片洼陷：溃烂范围广且深，提示火毒炽盛。
  • 久病溃烂
    • 淡红反复：溃烂反复发作，颜色淡红，提示虚火上炎。
    • 苍白不愈：溃烂苍白，久不愈合，提示气血肾阳亏虚。

伪膜

  • 松厚易拭
    • 表现：伪膜松散厚实，易擦去。
    • 病因：肺胃热浊，属于轻证。
  • 坚韧白喉
    • 表现：伪膜坚韧不易擦去，呈白喉状。
    • 病因：疫毒或热毒伤阴，属于重证。

3. 望颈项

望颈项

原理

  • 肝经：循行经过喉咙，肝郁痰凝可致瘿瘤。
  • 肾经：循行经过喉咙，肾虚可致项软或瘰疬。
  • 太阳经：行于项背，风寒袭太阳可致项强。

外形变化

  1. 瘿瘤（颈前肿块）

    • 病因：肝郁痰凝或水土失调。
    • 特征：肿块随吞咽移动。

  2. 瘰疬（颈侧串珠）

    • 病因：肺肾阴虚或风热时毒。
    • 特征：大肿块称“瘰”，小结节称“疬”，常呈串珠状。

动态变化

  1. 项强（颈部拘紧）

    • 风寒袭太阳：伴有头痛、恶寒。
    • 热极生风：伴有高热、抽搐。
    • 落枕：因睡姿不当或劳损导致。

  2. 项软（抬头无力）

    • 小儿：佝偻病或先天不足。
    • 久病：精气衰竭，属危重征象。

  3. 颈脉怒张

    • 心血瘀阻或肺气壅滞：颈脉明显隆起。
    • 心肾阳衰：水气凌心。

4. 望躯体

望胸胁

望胸胁原理：

  • 心肺：胸为宗气所聚之处，心肺功能失调可致胸廓异常。
  • 胃经：乳房属胃经，乳痈与胃热有关。
  • 肝胆经：胁肋属肝胆经，胁部异常与肝胆功能相关。

异常胸廓

  1. 扁平胸

    • 特征：胸部前后径缩小。
    • 病因：肺肾阴虚或气阴两虚。

  2. 桶状胸

    • 特征：胸部呈圆桶状。
    • 病因：伏饮积痰或肾不纳气。

  3. 鸡胸

    • 特征：胸骨前突。
    • 病因：先天不足或后天失养（如佝偻病）。

  4. 漏斗胸

    • 特征：胸骨凹陷。
    • 病因：先天发育不良。

  5. 肋如串珠

    • 特征：肋骨增厚呈串珠状。
    • 病因：肾精不足或后天失养。

  6. 胸不对称

    • 塌陷：肺痿或术后。
    • 膨隆：悬饮或气胸。

乳痈
乳痈是乳房红肿热痛的病证，多见于哺乳期。

  • 病因：肝郁胃热或外感邪毒。

望腹部

望腹部原理

  • 中下焦脏腑：腹部是中下焦脏腑所居之处。
  • 肝脾肾：腹部为肝脾肾气机枢纽，功能失调可致腹部异常。
  • 气血盛衰：腹部形态和静脉暴露反映气血盛衰。

腹部膨隆（高于胸耻连线）

  1. 水肿病
    • 特征：伴有周身肿胀。
    • 病因：肺脾肾失调或水湿内停。
  2. 鼓胀
    • 特征：四肢消瘦。
    • 病因：肝郁脾虚或气滞血瘀水停。

腹部凹陷（低于胸耻连线） 腹部凹陷根据病程和表现分为三种情况：

  1. 久病凹陷（舟状腹）
    • 病因：脾胃气虚或机体失养。
  2. 急病凹陷（津伤）
    • 病因：吐泻太过或津液大伤。
  3. 危候（肉消着骨）
    • 病因：脏腑精气耗竭。

腹露青筋（静脉暴露） 腹部青筋暴露是鼓胀重证的表现。

  • 病因：肝郁气滞、脾虚湿阻或脾肾阳虚、水停瘀阻。

望腰背

望腰背原理

  • 心肺：背为心肺之府，背部异常与心肺功能相关。
  • 肾：腰为肾之府，腰部异常与肾精亏虚有关。
  • 督脉：循行于脊柱，脊柱异常与督脉功能失调有关。

脊柱后突 脊柱后突主要表现为驼背（龟背）：

  • 小儿：肾精亏虚或后天失养。
  • 成人：脊椎疾患。
  • 背曲肩随：精气虚衰。

脊柱侧弯 脊柱侧弯呈弧形或S形：

  • 病因：坐姿不良、先天不足、肾精亏虚或胸疾。

脊疳 脊疳表现为脊骨锯齿状突出：

  • 病因：脏腑精气极亏。

腰部拘急 腰部拘急表现为活动受限、转侧不利：

  • 病因：
    • 寒湿侵袭导致经气受阻。
    • 跌仆闪挫导致血脉瘀滞。

5. 望四肢

望四肢

可观察四肢的外形和动态变化，判断脏腑功能、气血津液的盛衰以及病证的寒热虚实。

原理

  • 脾：主四肢肌肉，脾虚可导致肌肉失养。
  • 肝：主筋，肝虚可导致筋脉失养。
  • 肾：主骨，肾虚可导致骨骼发育不良。
  • 肺：朝百脉，肺虚可影响气血运行。

外形异常

四肢的外形异常主要包括肿胀、畸形和小腿青筋，具体如下：

  1. 肢体肿胀

    • 关节肿痛：提示湿热痹阻。
    • 全身浮肿：提示水肿病。
    • 象皮腿：提示丝虫病。

  2. 四肢畸形

    • 膝部肿大
      • 红肿热痛：提示热痹。
      • 鹤膝风：提示气血亏虚或寒湿。
      • 紫暗漫肿：提示外伤。
    • 下肢畸形
      • O型腿/X型腿：提示先天不足或后天失养。
      • 足内翻/外翻：提示发育不良。
    • 手指变形
      • 梭状指：提示风湿痰瘀。
      • 杵状指：提示心肺气虚或血瘀痰阻。

  3. 小腿青筋

    • 病因：寒湿血瘀或气虚负重。

动态异常

四肢的动态异常主要表现为肢体痿废，具体包括以下三种情况：

  1. 痿病

    • 病因：肺热、湿热、脾胃虚或肝肾亏。

  2. 半身不遂

    • 病因：中风。

  3. 截瘫

    • 病因：脊髓损伤。

6. 望二阴

望二阴

通过观察前阴（外生殖器）和后阴（肛门）的异常表现，判断脏腑功能、气血津液的盛衰以及病证的寒热虚实。

原理

  • 前阴相关：

    • 肾：主宰前阴。
    • 肝经：绕行前阴。
    • 冲任：联系胞宫。

  • 后阴相关：

    • 脾：主升降功能。
    • 大肠：主传导排泄。
    • 肾：司二便（大小便）。

前阴异常

前阴的异常表现包括阴囊肿大、阴部湿疹和子宫脱垂，具体如下：

  1. 阴囊肿大

    • 疝气：小肠下坠。
    • 瘀血或水液：导致睾丸肿胀。
    • 囊痈：表现为红肿如瓢，多由肝经湿热引起。

  2. 阴部湿疹

    • 急性：由肝经湿热合并风邪引起。
    • 慢性：由阴虚血燥导致，表现为苔藓样变化。

  3. 子宫脱垂（阴挺）

    • 病因：
      • 气虚下陷。
      • 冲任虚损。
      • 产育过多或胞络损伤。

后阴异常

后阴的异常表现包括肛裂、痔疮、肛瘘、脱肛和肛痈，具体如下：

  1. 肛裂（纵行溃疡）

    • 病因：热结肠燥或阴虚津亏，导致便秘用力过度。

  2. 痔疮（紫红肿块）

    • 类型：内痔、外痔、混合痔。
    • 病因：湿热蕴结或血热肠燥。

  3. 肛瘘（瘘管流脓）

    • 病因：痈肿余毒未清，导致溃口难以愈合。

  4. 脱肛（直肠脱出）

    • 病因：脾虚中陷，常因久泻或久咳引发。

  5. 肛痈（红肿流脓）

    • 病因：湿热下注或邪毒阻滞，可能进一步发展为肛漏。

7. 望皮肤

望皮肤

原理

  • 肺：主皮毛，卫气循行于皮肤，肺虚或外邪侵袭可导致皮肤异常。
  • 经络：外荣气血，气血失和可引发皮肤病变。
  • 肾：肾阳虚衰与黑疸相关，肾精不足影响皮肤营养。

色泽异常

  1. 皮肤发黄（黄疸）

    • 阳黄：湿热蕴蒸，皮肤呈现橘黄色。
    • 阴黄：寒湿阻遏，皮肤呈现烟熏色。

  2. 皮肤发赤（丹毒）

    • 抱头火丹：风热化火引起。
    • 流火：湿热化火引起。
    • 赤游丹：游走不定，病因多为风热或湿热。

  3. 皮肤发黑（黑疸）

    • 黄疸后期：劳损伤肾，导致皮肤发黑。
    • 全身发黑：肾阳虚衰所致。

  4. 皮肤白斑（白驳风）

    • 病因：风湿侵袭，导致血不荣肤，出现白斑。

形态异常

  1. 皮肤干枯

    • 病因：阴津耗伤、营血亏虚或燥邪侵袭，导致气血滞涩，皮肤失去滋润。

  2. 肌肤甲错

    • 表现：皮肤粗糙，状若鱼鳞。
    • 病因：血瘀日久引起。

  3. 肌肤水肿

    • 阳水：眼睑先肿，多由外感风邪引起。
    • 阴水：下肢先肿，多由脾肾阳衰引起。

皮肤病症

原理

  • 卫气不固/外邪侵袭：外邪侵入皮肤，导致斑疹、水疱等病变。
  • 营血热毒/气血失和：营血内热或气血不调，导致疮疡、痤疮。
  • 脏腑湿热/痰瘀凝滞：湿热、痰瘀内生，使皮肤病变加重。

斑疹

斑疹是皮肤上常见的病变，分为斑和疹两种：

  1. 斑（平铺不褪色）

    • 阳斑：热入营血，表现为红紫成片。
    • 阴斑：气不摄血，表现为淡红隐现。

  2. 疹（突起褪色）

    • 麻疹：时邪疫毒引起，表现为桃红麻粒。
    • 风疹：风热时邪引起，表现为淡红斑丘。
    • 瘾疹：风邪或过敏引起，表现为骤起骤退。

水疱

水疱是皮肤上充满液体的疱疹，具体包括以下五种类型：

  1. 水痘（晶莹顶满）

    • 病因：时邪湿热，多见于儿科传染病。

  2. 白㾦（晶疹/枯疹）

    • 晶疹：津气充足，属顺证。
    • 枯疹：津气亏竭，属逆证。

  3. 热气疮（口鼻阴部）

    • 病因：肺胃湿热或肝经湿热。

  4. 缠腰火丹（带状水疱）

    • 病因：肝经湿热，表现为单侧刺痛。

  5. 湿疹（糜烂渗液）

    • 病因：湿热风邪，常与皮肤过敏相关。

疮疡

疮疡是皮肤上的化脓性病变，具体包括以下四种类型：

  1. 痈（红肿高突）

    • 病因：湿热火毒，属阳证，易溃脓。

  2. 疽（深部扩散）

    • 有头疽：火毒蕴结，属阳证。
    • 无头疽：寒痰凝滞，属阴证。

  3. 疔（钉状深硬）

    • 病因：疫毒火毒，毒邪易走散。

  4. 疖（浅表小肿）

    • 病因：火热毒邪，易脓易愈。

经方讲解：

  • 葶苈大枣泻肺汤、苇茎汤，治疗肺痈，症见咳吐脓血，气喘。
  • 麻黄连翘赤小豆汤，治疗瘾疹（荨麻疹），症见皮肤起疹，热象。
  • 半夏泻心汤，治疗痤疮，症见皮肤长痘，寒象、热象兼有。
  • 理中汤，治疗流涎，症见流口水，寒象。
  • 苓桂术甘汤，治疗肠鸣音亢进，症见肠鸣音，寒象。

经验方：

  • 瓜蒌红花汤，治疗缠腰火丹（带状疱疹）。

常用合方：

  • 桂枝汤合玉屏风散，治疗鼻鼽（过敏性鼻炎），症见反复流清涕，频繁打喷嚏，寒象。有脾虚表现，合补中益气汤。

1. 舌的结构

舌的基本分区

  1. 舌的上下：

    • 舌面（舌背）：舌的上方。
    • 舌底：舌的下方。

  2. 舌的前中后：

    • 舌尖：舌的前端。
    • 舌中：舌的中部。
    • 舌根：舌体的后部，人字形界沟之前。
    • 舌边：舌体两侧。

  3. 舌底的结构：

    • 舌系带：舌底正中线的连襟。
    • 舌下肉阜：舌系带终点两侧的小圆突起，为腺管开口。
    • 金津（左）、玉液（右）：舌下肉阜的中医称谓，为胃津、肾液上潮的通道。

舌的表面结构

  1. 舌正中沟：舌背正中纵行皱褶，不甚明显。
  2. 舌乳头（分四种）：
    • 丝状乳头（最多）：遍布舌背前2/3，呈圆锥状，白色，易角化脱落，形成舌苔（与舌象关系密切）。
    • 蕈状乳头（分散）：位于舌尖及舌边，呈红色小点，富含血管（影响舌质变化）。
    • 轮廓乳头（少数）：较大，位于舌根与舌体交界处（与味觉相关）。
    • 叶状乳头（最少）：位于舌体后部两侧边缘，人类已退化，仅新生儿明显（与味觉相关）。

2. 舌诊原理

舌与脏腑的联系，舌与脏腑通过 经络 相连，其中 心、脾胃 关系最密切。

  1. 舌为心之苗窍（手少阴心经）

    • 《灵枢·脉度》：“心气通于舌，心和则舌能知五味矣。”
    • 心主血脉 → 舌脉络丰富，心血上荣 → 舌色反映气血情况。
    • 心主神明 → 影响舌的运动、语言清晰度 → 舌运动异常，提示神志病变。

  2. 舌为脾之外候（足太阴脾经）

    • 《灵枢·脉度》：“脾气通于口，脾和则口能知五谷矣。”
    • 脾开窍于口，主运化、化生气血。
    • 舌苔 由 胃气熏蒸谷气上承 而成 → 反映脾胃功能。
    • 章虚谷：“脾胃为中土，邪入胃则生苔，如地上生草也。”

  3. 其他脏腑与舌的联系

    • 肝（足厥阴肝经） → 藏血，主筋，经络连舌本。
    • 肾（足少阴肾经） → 藏精，经络夹舌本。
    • 膀胱（足太阳膀胱经） → 经筋结于舌本。
    • 肺（肺系） → 上达咽喉，与舌根相连。
    • 其他脏腑 也通过 经络沟通，影响舌象。

舌象——脏腑气血的窗口

  • 脏腑病变 → 舌象变化
  • 观察 舌质、舌苔，可判断 气血盛衰、脏腑功能状态。

4. 舌诊方法与注意事项

1. 舌诊的方法操作步骤

  • 医者站位： 视线略高于患者，俯视口舌部位。
  • 患者姿势： 坐位/仰卧，头稍扬，口微张，舌自然伸出，舌尖略向下。
  • 望舌顺序： 先舌尖→舌中→舌边→舌根。
  • 观察顺序： 先舌质（色、光泽、形态、动态），再舌苔（有无、色、质、分布）。
  • 控制时间： 避免长时间伸舌，影响舌质观察，可短暂休息后再望舌。
  • 辅助检查：
    • 刮舌验苔： 刮舌板自舌根向舌尖刮3~5次，看苔是否能去除。
      • 刮不去/留污质 → 里实邪
      • 刮去后舌光滑 → 虚证
    • 揩舌检查： 以湿棉签揩舌，辨别染苔。
    • 舌下络脉： 观察络脉充盈情况。
    • 问诊： 了解 味觉、舌痛、麻木、灼辣感、运动异常。

2. 舌诊的注意事项

  • 避免光线干扰
  • 避免饮食和药物干扰
  • 避免口腔环境影响
  • 伸舌姿势的影响

5. 舌诊内容

舌诊的内容

  1. 望舌质（察脏腑、辨气血）
  • 舌神：有神则脏腑功能正常，失神则病重。
  • 舌色：淡红、红、绛、紫、青白等，反映气血状况。
  • 舌形：大小、厚薄、胖瘦、裂纹、齿痕等。
  • 舌态：伸缩灵活度、强硬、痿软、颤动等。
  1. 望舌苔（辨病位、测邪正）
  • 苔色：白、黄、灰、黑，反映邪气性质。
  • 苔质：薄厚、润燥、腻腐、剥脱，反映病位深浅。
  1. 望舌质与苔，需综合分析，才能正确判断病情。

正常舌象 标准：淡红舌，薄白苔

  • 舌质：淡红色，大小适中，柔软灵活，光泽润泽。
  • 舌苔：薄白均匀，干湿适中，不黏不腻，刮之不去，苔下有根。
  • 生理意义：胃气旺盛，气血充盈，脏腑功能正常。

舌象的生理性变异

  1. 受外界环境影响
  • 季节影响：
    • 秋季：燥气当令，苔偏薄、偏干。
    • 冬季：寒气盛，舌常湿润。
    • 夏季：暑湿重，舌苔多厚，或淡黄。
  • 地域影响：
    • 东南潮湿炎热 → 舌苔易厚腻。
    • 西北、东北寒冷干燥 → 舌偏红，苔少或干。
  • 意义：若舌象符合生理变异，但无不适症状，多为正常，否则需警惕疾病前兆。

受个体因素影响

  • 年龄因素：
    • 儿童：脾胃稚嫩、代谢旺盛，舌淡嫩，苔少易剥。
    • 老年：气血衰退，舌暗红，易有齿痕。
  • 性别因素：
    • 女性月经期：舌乳头充血，舌红点增多，经后恢复。
  • 体质因素：
    • 肥胖者：舌胖大，色淡，多齿痕。
    • 消瘦者：舌瘦小，色偏红。
    • 特殊舌象：裂纹舌、地图舌、齿痕舌等，多为先天因素，若无其他病理变化，则无诊断意义。

6. 望舌质

观察 神、色、形、态。

望舌神

  1. 荣舌（有神）

    • 特征：舌 红润、有光泽，活动灵活。
    • 意义：气血充盛，病中见之为善候。
    • 机理：气血旺盛，上荣于舌。

  2. 枯舌（无神）

    • 特征：舌 干枯无光，活动不灵。
    • 意义：气血衰败，多见于重症。
    • 机理：气血败坏，不能荣润舌体。

望舌色

舌色 特征 临床意义 机理分析
淡红舌 淡红润泽 健康人、病轻、外感表证 气血调和，胃气旺盛
淡白舌 苍白无血色 气血两虚、阳虚 血不荣舌，阳虚运血无力
红舌 鲜红或有芒刺 热证（舌尖红—心火、舌边红—肝火） 血热加速流动，或阴虚火旺
绛舌 深红、暗红 热盛证（温热入营、久病阴虚） 热入营血，血浓缩或虚火旺盛
青紫舌 青紫、瘀点 气血瘀滞（瘀血、寒凝、气虚） 血行不畅，舌络瘀阻

望舌形
舌形主要观察舌质形态，包括 老嫩、胖瘦、点刺、裂纹、齿痕 等特征。

  1. 老嫩舌（标志虚实）
  • 老舌（实证）：纹理粗糙、苍老、色暗
  • 嫩舌（虚证）：纹理细腻、浮胖娇嫩、色浅
  • 机理：实邪亢盛→舌苍老；气血不足/阳虚→舌嫩色淡
  1. 胖瘦舌（标志气血津液状态）
  • 胖舌（水湿痰饮）：舌体胖大或肿胀满口
    • 淡胖 → 脾肾阳虚，水湿停滞
    • 红胖 → 脾胃湿热、痰热
    • 肿胀 → 湿热、热毒、酒毒、血瘀
  • 瘦舌（气血不足）：舌体瘦小
    • 淡瘦 → 气血两虚
    • 红瘦、少苔 → 阴虚火旺
  1. 点刺舌（标志热盛）
  • 点：红点、白点、黑点
  • 刺：舌乳头增大如芒刺，多见于舌尖、舌边
  • 主病：脏腑热盛、血分热极
  • 颜色辨病：
    • 红点刺 → 气分热盛
    • 鲜红点刺 → 血热、阴虚火旺
    • 绛紫点刺 → 热入营血、气血壅滞
  • 部位辨病：
    • 舌尖点刺 → 心火亢盛
    • 舌边点刺 → 肝胆火盛
    • 舌中点刺 → 胃肠热盛
  1. 裂纹舌（标志阴血亏虚、脾虚湿侵）
  • 红绛裂纹 → 热盛伤阴
  • 淡白裂纹 → 血虚不润
  • 淡白胖嫩+齿痕+裂纹 → 脾虚湿侵
  • 先天性裂纹舌（无病理意义）
  1. 齿痕舌（标志脾虚、湿盛）
  • 胖大+润+齿痕 → 寒湿壅盛、阳虚水湿
  • 淡红+齿痕 → 脾虚或气虚
  • 红肿满口+齿痕 → 湿热痰浊
  • 淡红嫩+轻齿痕 → 先天性齿痕舌，病情较轻

望舌态
舌态指舌的动态表现，正常为 伸缩自如，灵活运动，异常舌态提示 脏腑失调，气血异常。

  1. 痿软舌（气血俱虚、阴亏已极）
  • 淡白无华 → 气血俱虚
  • 红绛少苔/无苔 → 热极伤阴、阴虚火旺
  • 机理：气血亏虚/阴液亏损→肌筋脉失养→舌痿软
  1. 强硬舌（热入心包、热盛伤津、风痰阻络）
  • 红绛少津+强硬 → 邪热炽盛
  • 胖大+厚腻苔+强硬 → 风痰阻络
  • 强硬+语言塞涩+麻木眩晕 → 中风先兆
  • 机理： 热邪扰心→舌强硬，津亏→筋脉失养→舌强，风痰阻络→舌不灵活
  1. 歪斜舌（中风或先兆）
  • 舌体向一侧偏斜
  • 色紫红+急性 → 肝风内动（风痉）
  • 淡红+缓慢 → 中风偏枯
  • 机理：肝风夹痰/瘀血阻络→一侧舌肌弛缓→舌歪
  1. 颤动舌（肝风内动）
  • 淡白颤动 → 血虚动风
  • 绛红颤动 → 热极生风
  • 红少津颤动 → 阴虚动风、肝阳化风
  • 酒毒内蕴 → 舌颤
  • 机理：气血亏虚→筋脉失养→舌无力→颤，阴虚风动→舌不稳→颤
  1. 吐弄舌（心脾有热）
  • 吐舌（舌伸出口不收回）→ 疫毒攻心、正气欲绝
  • 弄舌（舌微露后立即收回，或舔唇、乱动）→ 热甚动风、小儿智力低下
  • 机理：心火旺/脾热→舌躁动
  1. 短缩舌（寒凝、痰阻、血虚、津伤）
  • 淡白/青紫+湿润 → 寒凝、气血俱虚
  • 胖+苔滑腻 → 痰浊阻滞
  • 红绛干燥 → 热盛伤津
  • 机理：寒→筋脉收缩，舌短；血虚→舌失濡养，舌短

7. 望舌苔

基本概念

  • 舌苔由胃气熏蒸胃中谷气、食浊凝聚而成。
  • 正常舌苔：薄白均匀、干湿适中，舌中部及舌根稍厚。

望苔质

1.薄苔 vs. 厚苔

  • 定义：
    • 薄苔（见底苔）：透过苔可隐见舌质
    • 厚苔（不见底苔）：遮盖舌质
  • 临床意义：
    • 薄苔：多见于病邪初起、邪在表，或正常表现（薄白苔）。
    • 厚苔：提示邪气入里、痰饮食积，病情较重。
  • 机理：
    • 正常时，薄白均匀提示胃有生发之气；由薄转厚（舌苔长）表示邪气渐盛、病情进展；由厚转薄（舌苔消）则说明正气胜邪或病退。

2.润苔 vs. 燥苔（及滑、糙苔）

  • 定义：
    • 润苔：苔泽润、有津、干湿适中
    • 滑苔：水分过多、摸之湿滑（多见于水湿、痰饮）
    • 燥苔：干枯、无津，有时伴干裂
    • 糙苔：颗粒粗糙如砂石
  • 临床意义：
    • 润苔：提示津液充足（正常或风寒、初起湿证）
    • 燥苔：提示津液受损，常因邪热、过汗、吐泻或痰饮、瘀血阻碍津液输布
  • 机理：
    • 舌苔从润转燥表示热重津伤；相反，从燥转润说明热退津复或饮邪转化。

3.腻苔 vs. 腐苔（及脓腐苔）

  • 定义：
    • 腻苔：颗粒细腻、致密，似涂油般紧贴舌面，刮之难去
    • 腐苔：颗粒疏松、粗大、厚重，形似豆腐渣，易刮去
    • 脓腐苔：黏厚如疮脓
  • 临床意义：
    • 均主痰浊、食积；脓腐苔则提示内痈、邪毒较重。
  • 机理：
    • 多因湿浊内蕴、阳气受阻，使得湿痰、食积停滞于舌面。

4.剥（落）苔

  • 定义：舌苔在疾病过程中部分或全部脱落，剥落处光滑无苔。
    • 分为：前剥（舌前部）、中剥（舌中部）、根剥（舌根部）、花剥（多处剥落）、地图舌（不规则剥落，边缘凸起）、镜面舌（全剥，光洁如镜）、类剥苔（剥落处尚有新生颗粒）。
  • 临床意义：
    • 主胃气不足、胃阴损伤或气血两虚。
  • 机理：
    • 因胃气乏或胃阴损伤，不能上承舌面；剥落范围大小与气阴、气血亏虚程度相关。
    • 需与先天性剥苔（常位于舌中央人字界沟前、呈菱形）鉴别。

5.偏苔 vs. 全苔

  • 定义：
    • 全苔：苔遍布整个舌面。
    • 偏苔：苔只分布于舌前、后、左或右某局部。
  • 临床意义：
    • 全苔多提示邪气散漫（常见湿痰中阻）；偏苔则常提示该部位对应的脏腑有邪气停聚。
  • 机理：
    • 如舌尖偏苔提示热在上焦；舌中或舌根偏苔提示胃肠或下焦问题。

6.眞苔 vs. 假苔

  • 定义：
    • 眞苔：坚敛着实、紧贴舌面，刮之难去，似自生在舌上，具有“根”。
    • 假苔：不着实，浮于舌面，刮之即去，无明显根蒂。
  • 临床意义：
    • 对判断疾病轻重、预后具有重要意义。
  • 机理：
    • 眞苔为胃气充足生成，若病中见厚，表明邪气深重但正气尚存；
    • 无根苔提示脾、胃、肾之气虚，正气衰竭。

望苔色

舌苔颜色主要分为白、黄、灰黑三类，临床需结合苔质、舌色、舌形及舌态综合判断。

1.白苔

  • 特征：舌苔呈白色，可分薄白（透见舌质）和厚白（遮盖舌质）。
  • 临床意义：
    • 薄白苔常为正常或表证、寒证；
    • 厚白苔可能提示湿浊、痰饮或食积。
  • 机理：
    • 白苔为舌苔本色，其干湿和厚薄变化反映外感寒湿或风热等状态；特殊时（如积粉苔、燥裂白苔）也可见于热证。

2.黄苔

  • 特征：舌苔呈黄色，根据程度可分为浅黄、深黄和焦黄。
  • 临床意义：
    • 主热证、里证。
    • 薄黄提示热势轻（如风热表证或风寒化热初起）；
    • 深黄、焦黄提示热邪较重。
  • 机理：
    • 邪热熏蒸导致苔色由白转黄，黄苔出现的位置（舌尖、舌中、舌根）还可指示热邪所在病位。

3.灰黑苔

  • 特征：颜色介于灰与黑之间，常见于舌中、人字界沟附近较深，向舌尖逐渐变浅。
  • 临床意义：
    • 主阴寒内盛或里热炽盛，均属重证。
    • 黑色越深，病情越严重。
  • 机理：
    • 灰黑苔可由白苔或黄苔转化而成。
    • 在寒湿病中通常伴湿润多津，在热性病中则干燥无津。
    • 部分患者因外界染色（如吸烟）也可出现，但需结合其他表现判断。

8. 舌诊意义及危重舌诊法

舌诊意义

  • 分辨病位浅深
  • 区别病邪性质
  • 判断邪正盛衰
  • 分析病势进退
  • 推测病情预后

前人总结的危重舌象：

  • 猪腰舌 舌面无苔，如去膜的猪腰。多见于热病伤阴，胃气将绝，主病危。
  • 镜面舌 舌深绛无苔而光亮如镜，主胃气、胃阴枯涸;舌色眺白如镜，毫无血色，也称眺白舌，主营血大亏，阳气将脱，均属病危难治。
  • 砂皮舌 舌粗糙有刺，如沙鱼皮，或干燥枯裂。主津液枯竭，病危。
  • 千荔舌 舌敛束而无津，形如干荔肉。主热极津枯，病危。
  • 火柿舌 舌如火柿色，或色紫而干晦如猪肝色。主内脏败坏，病危。
  • 赭黑舌 舌质色赭带黑。主肾阴将绝，病危。
  • 瘦薄无苔舌 舌体瘦小薄嫩，光而无苔。属胃气将绝，难治。
  • 囊缩卷舌 舌体卷缩，兼阴囊缩入。属厥阴气绝，难治。
  • 舌强语塞 舌体强道，转动不灵，且语言塞涩。多属中风痰瘀阻络，难治。
  • 蓝舌而苔黑或白 舌质由淡紫转蓝，舌苔由淡灰转黑，或苔白如霉点、糜点。主病危重，难治。

4. 望小儿食指络脉

原理及意义

  • 基本原理

    • 小儿食指掌侧前缘的浅表络脉（又称“指纹”）是寸口脉的分支，属于手太阴肺经。
    • 与诊寸口脉具有相同的诊断意义，可反映体内病变。

  • 临床意义

    • 小儿3岁以内脉诊因寸口脉短小、易哭闹而受限，望指纹能弥补脉诊不足，辅助判断儿科疾病。

望诊方法

  1. 抱姿及观察环境

    • 抱小儿时让其面向光亮。

  2. 操作步骤

    • 用医生左手拇指和食指握住小儿食指末端；
    • 用右手拇指侧缘自指尖向指根部轻推几次，力度适中，使络脉充分显露。

小儿食指正常络脉

  • 分关描述（按指节划分三关）：

    • 风关（第一节）：掌指横纹至第二节横纹之间
    • 气关（第二节）：第二节横纹至第三节横纹之间
    • 命关（第三节）：第三节横纹至指端

  • 正常特点

    • 指纹纹色浅红、红黄相间，隐隐可见于风关内，粗细适中。

  • 影响因素

    • 年龄：幼儿络脉显露较长，年长儿则略短。
    • 体质：皮肤薄嫩易显，皮肤厚、肥胖者则模糊。
    • 气候：天热时络脉扩张（增粗、变长），天冷时收缩（细缩）。

小儿食指病理络脉

观察重点：深浅、颜色、形态和长短
其辨证要点可概括为“浮沉分表、红紫辨寒热、淡滞定虚实、三关测轻重”。

  1. 浮沉分表里

    • 浮显：指纹浮于皮肤表面，表明病邪在表（外感表证）。
    • 沉隐：指纹沉于皮下不易见，提示病邪在里（内伤里证）。

  2. 红紫辨寒热

    • 色鲜红：主外感风寒表证（气血外浮）。
    • 紫红：提示内热盛，邪热内陷。
    • 其他颜色：
      • 青色：多见于疼痛、惊风或肝风内动；
      • 淡白：主脾虚、疳积；
      • 紫黑：提示血络郁闭，多属病危。

  3. 淡滞定虚实

    • 指纹浅淡纤细：气血不足、虚证。
    • 指纹浓滞增粗：邪正相争，气血壅滞，属实证。

  4. 三关测轻重

    • 风关：指纹显露于风关，邪气浅、病情轻，多见于外感初起。
    • 气关：指纹达于气关，提示邪气入经、病情较深。
    • 命关：指纹达至命关，说明邪气已入脏腑，病情严重；若直达指端（“透关射甲”），预后凶险。

5. 望排泄物

基本原理

  • 范围：观察人体的分泌物（如汗、泪、涕、唾、涎）、排泄物（大便、小便）及病理产物（痰、呕吐物、脓血等）。
  • 关系：各种排出物的产生和形成与脏腑功能密切相关，反映机体寒热、虚实状态。
  • 总规律：
    • 颜色淡、白、质稀者，多属虚证、寒证
    • 颜色深、黄、质稠者，多属实证、热证

望痰

  • 定义：痰为肺及气道排出的病理性黏液。
  • 观察要点：
    1. 痰白、质清、稀：常见于寒痰（寒邪阻肺或脾阳不足，湿聚为痰）。
    2. 痰黄、质黏稠（甚至结块）：提示热痰（邪热煎津形成）。
    3. 痰少、质黏，不易咯出：多见于燥痰（燥邪犯肺或肺阴虚）。
    4. 痰中带血（色鲜红）：称为咯血，见于肺痨、肺癌，多因肺阴亏虚或肝火犯肺。
    5. 咯吐脓血痰，味腥臭：为肺痈，提示热毒蕴肺、肉腐成脓。

望涕

  • 定义：涕是鼻腔分泌的黏液，为肺之液。
  • 观察要点：
    1. 新病流清涕、鼻塞：多见于外感表证（风寒）。
    2. 鼻塞流浊涕：提示风热表证。
    3. 反复大量清涕，伴鼻痒、喷嚏：常因肺气虚、卫表不固。
    4. 久流浊涕，质稠、量多、带气腥臭：提示湿热内蕴，形成鼻渊。

望涎唾

  • 定义：涎唾包括口腔分泌的清稀（涎）与黏稠、泡沫状（唾）液体，分别反映脾与肾功能及与胃有关。
  • 观察要点：
    1. 口流清涎、量多：常见于脾胃虚寒（脾胃阳虚，气不化津）。
    2. 口中反复吐黏涎：多因脾胃湿热，湿邪困阻中焦。
    3. 小儿口角流涎、颐下湿渍：称为“滞颐”，提示脾虚不能摄津。
    4. 睡中流涎：可能与胃内热或宿食内停、痰热内蕴有关。
    5. 频繁吐唾：或因胃中虚冷、肾阳不足，水液上泛，或因宿食、湿邪留滞引起。

望呕吐物

  • 定义：呕吐是胃气上逆的表现，可为饮食物、清水、痰涎，或混有脓、血等。
  • 观察要点：
    1. 呕吐物清稀、无酸臭：多属寒呕，见于脾胃阳虚或寒邪犯胃。
    2. 呕吐物秽浊、有酸臭：提示热呕，因邪热犯胃、蒸腐胃中饮食。
    3. 呕吐清水、痰涎：提示水饮内停，脾运不健。
    4. 呕吐不消化、气味酸腐的食物：常见于伤食，暴饮暴食损伤脾胃。
    5. 呕吐黄绿苦水：主肝胆湿热或郁热。
    6. 吐血（暗红或紫暗，有块、夹食物残渣）：提示胃热积、或肝火犯胃、血瘀等。

结语

中医望诊乃四诊之首，以"司外揣内"为道，观神色形态之变，察五色应五脏之象。青赤黄白黑映肝心脾肺肾，荣枯明晦辨气血盛衰之机。以目观象，以象测藏，藏于内者必形于外，此乃"有诸内必形诸外"之真谛。医者凝神静气，见微知著，从皮毛肌腠至五官九窍，皆可窥天地人三才相应之妙。望诊之道，既是医术，更是参透天人相应、形神合一的生命智慧。生命万象，皆可映于方寸皮脉之间。
